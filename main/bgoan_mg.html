१ रीधपुर प्रांती अवतारु: स्वीकारु

  • रीधपुर प्रांतीं दीढा गाउवा कांटसरें : एकी वासना : कांटोपरें : एकी वासना : माउरें : काणवां ब्राह्मणांचां घरीं गर्भी अवतारु स्वीकरीला : अनंतनायकु पीता : नेमाइसें माता : गोसावियांचीये मातेसि बहुतें लेंकरुवें जालीं : तीयें जीतिचि नां : तेणें दुखें थोर आरंबळली : मग गोसवी नीपटनेयांचे जाले : गुंडो ऐसें नांव ठेविलें : वरीखा दीसा माता-पीता अपरोक्ष जालीं : मग मामेनि माउसीयां वाढविलें : सातां वरिखां मुंजिबंधन जालें : सीका-सुत्र कासोटी दीधली : पळसुला : ऐसे च्यार्‍ही दीस जाले : चौथां दीसीं उपाधीं म्हणीतलें : “गुंडो ! वों भवति भीक्षां देही : येणें शब्दें भीक्षा माग :” म्हणति ‘भीक्षां देही’ ऐसें म्हणति : परि वोंकारु न म्हणति : ऐसें मुंजीव्रत जालें : मग रीधपुरासि घेउनि आलीं : ॥१॥

    • ३) मृत्य बालक जीववणें

  • ८ मृत्य बाळक जीववणें एकाचें लेंकरु गोसावी खेळवीति : तें सरलें : तें नेउनि वाजेस्वरा इसान्य कोंणीं पांनदीं पुरीलें : तवं गोसावीं तयाचेया घरासि बीजें केलें : तवं गोसावीयातें देखिलें : आणि रडों लागलीं : मग तीही म्हणीतलें : “राउळो ! जिया लेकरुवातें तुम्हीं खेळवा तें आजि सरलें :” म्हणौनि अवघीं रडों लागलीं : आंणि राउळांसि अनुकुंपा उपनली : मग गोसावी म्हणीतलें : “आवो मेला जाये : आणिं आंणी नां म्हणे :” तेहीं म्हणीतलें आतां काइ आंणींन राउळो ? ” “आवो मेली जाये : आंणि आंणी आंणी नां म्हणे :” तीहीं म्हणीतलें : “आतां काइ आंणीन राउळो ?” “आवो मेली जाये : आंणांवें म्हणे : नांणांवें म्हणे : आवो आंणांवेंचि म्हणें :” म्हणौनि दुडुदुडु-करि धांवां घेतली : तेथ बीजें केलें : दगड फेडीले : उकरीलें : मग काढीलें झाडीलें : पुसीलें : आनंदभरीत वास पांहों लागलें : मग उपंगळीए घेउनि आले : तेहीं देखीलें : आणि कवाड घातलें : “नेया नेया राउळो ! सव परते नेया :” तवं तयांचां सेजारीं म्हांतारी एकि होती : तीचीए उपंगळीये नेउनि घातलें : तीया म्हणीतलें : “राउळो ! मीं काइ करींन ? मीं सतानीक बाइल :” मग गोसावीं तीची चुची तयांचां तोंडीं घातली : आंणि त्रहाटीत पान्हावो आला : तें चुटचुटां पेवों लागलें : तवं तीहीं कवाड उघडीलें : पाहाति तवं तैसें देखीलें : तवं गोसावीं बीजें केलें : तीयें मागों लागलीं : म्हातारीया म्हणीतलें : “तुम्हासि देत होते तेहवळिं कां नेघाचि ? मां आतां मी कां देइन ?” तैसीचि महाजनांपासि न्यावो सांघावेया नीगालीं : महाजनीं म्हणीतलें : “राउळीं इसि दीधलें : तें लेंकरुं इचेंची : तुम्हां लोभ करुं लाभे परि लेंकरुं न लभे : आतां तुं ऐसें करि : याचां सेजारीं अस :” मग तयाचां सेजारीं असों लागली : मग तेहीं बाज-सुपवती दीधली : एकि गाए दीधली : मग तो सतानीकु जाला : गोपि प्रवीणू जाला : तया नांवं दामोधरु : जवं जीयाला तवं तयासि जरु तावो नांहीं : आपुलीए अनुभवीचीया केलि लीळा भक्तिजनापुढें सांघे : ॥८॥

    • ४) लाकांची भाकरि म्हणणे

  • २९ लांकांची भाकरि म्हणणें तेळीणीचे लेंकरुं लांकांची भाकरि खात होतें : तवं तेथ गोसांवीं बीजें केलें : तयाचीए हातीची भाकरि हीरतीली : ते गोसांवी खावों लागले : तेळीणी म्हणीतलें : “राउळो ! राउळो ! हे भाकरि तेलीयांची :” गोसांवी म्हणीतले : “ आवो ! मेली जाए : तेलीयाची नव्हे म्हणे : हे लांकांची भाकरि :” म्हणुनी श्रीमुखीं घांसु घेउनि मग बीजें केलें : ॥२९॥

    • ५) सुनां दुध वाढणे

  • ३८ सुनां दुध वाढणें वीठलु जोइसी : तयातें दोघी सुनां : तयांते गाइ दुभेति : दुध अपार नीगे : तयांचे घर एमाइसांचेया घरा सेजेंचि : तया नांवं एल्हाइसें : सासु ते दुधेंसीं जेवी : आणि सुनांसि ताक वाढी : आणि तीया मनोर्थ करीति : तवं गोसांवी बीजें करीति : तीया रात्रीं जेउं बैसलीया : आळंदा दुध नीववीते असेति : तें दुध सुनांसि गाडुगेवरि वाढीति : तीया वरिपडौनि म्हणेति : “ राउळो ! राउळो! सवळे ताकवणीं होआंवे असें :” ऐसें मावेंचे लटकें मणेति : भीतरि सुख होए : गोसांवी म्हणति : “ मेली जाए : बोबाए नां : ” म्हणौनि कोपु नटले : आणि तीया उगीयाचि घेति : सासुवेसि ताक वाढीति : आंणि बीजें करीति : तवं सवळेचि गुसळीति तवं ताक होए : आणि लोणीं जेतुकें तेतुकेंचि होये : ॥३८॥

    • ६) गुढयां श्रीचरण लावणें

  • ७४ गुढयां श्रीचरण लावणें गोसावीं एकांकी राज्य करीत होते : तै एतां जातां मार्गीं गुढेनसीं खेळु करीति : श्रीचरण लावीति : आंणि म्हणेति : आवो हा आजि यथ असे म्हणे : हा असेचि म्हणे :” ऐसा खेळु करीति : मग बीजें करीति : मग गोसावीं राज्यधर्म स्वीकरीलयाउपरि दांडीए बैसेति : “ आवो डोळो डोळो :” म्हणेति : आंणि भक्तिजनीं दांडी मेळवीली : मग दांडीयवरि बैसेति : आंणि बीजें करीति : तो ठावो पावति : आंणि तीए वेगळा श्रीचरणु काढीति : लोंबतु घालीति : मग हालतु वांकीचा श्रीचरणु गुढयासि लावीति : (बाहुल्यें आंगठाचि लावीति :) ऐसा खेळु करीति : मग बीजें करीति :॥७४॥

    • ७) माहादाइसाची सेवा स्वीकार

  • ८७ माहादाइसाची सेवा स्वीकारू गोसावेयांसि उदर-वेथेची प्रवृत्ति जाली: “ आवो! एथ दुखताए म्हणे : आवो ! दुखताए म्हणें : ” मग माहादाइसीं दीवयावरि हातु सेकिला : आंणि बोंबीएवरि ठेविला : “ आवो! एथचि ओ: एथचि ओ : हाए : ” म्हणौनि स्वीकरीलें : मग माहादाइसीं म्हणीतलें : “ गोसावियांसि सेकु मानतु असे : तरी सगडी करूं : ” मग माहादाइसी सगडी केली : मग माहादाइसी पोतौडी सेकीति :आणिं उदरावरि ठेविति : आंणि “हाए” म्हणौनि स्वीकरीति आंणि म्हणेति : “ आवो ! एथचि ओ ! एथचि ओ !” म्हणौनि श्रीमुख पांगुरवीति : मागुतें उघडीति : उदक मागेति : ते माहादाइसे वोळगवीति : ऐसा सेकु केला : मग नीकें जालें : ॥८७॥

    • ८) माहादाइसाचें धीडरें आरोगणे

  • ८८ माहादाइसाचें धीडरें आरोगणें एकु दीं माहादाइसीं गोसावियांतें वीनवीलें : “जी जी ! मीं आजि गोसावियांसि धीडरें वोळगवीन जी ! : गोसांवी खेळावया बीजें न करावें जी !” गोसांवीं वीनवणी स्वीकरीली : “आवो मेली जाए : धीडरें देइल म्हणे :” म्हणौनि हरीखु स्वीकरीला : खेळों बीजें करावें ते न करीतीचि : “आवो धीडरें देइल म्हणें : खांवें : न खांवें म्हणे :” मग माहादाइसीं धीडरें केलें : मग ताट केलें : आसन रचीलें : गोसांवी आसनीं उपवीष्ट जाले : मग माहादाइसीं धीडरें ओळगवीलें : वाटीए तुप घातलें : मग गोसांवीं धीडरें अवलोकीलें : आंणि म्हणीतलें : “आवो हें धीडरें नव्हे म्हणे : हा आहीता म्हणे : आवो धीडरें आंणि आंणि ना म्हणे :” म्हणौनि कोपु नटले : माहादाइसीं म्हणीतलें : “जी जी ! आमतें गंगातीरीं धीडरें म्हणेति : तुमतें वराडीं आहीता म्हणेति :” “आवो : धीडरें आंणि आंणि नां म्हणे :” म्हणौनि कोपु नटले : माहादाइसें वीचारूं लागली : आंणि स्फुरलें : मग दुधा आंतु सोजी घातली : ते कालवीली : भीडेयांसि तुपाचा बोंळा दीधला : वरि लींगाकृत घातलें : एकी वासना : संकाकृत घातलें : वरि एळां मीरीयां-लवंगाची पुडी घातली : एकी सवा जालें : मग दुसरीएकडे उलथीलें : आंणि काढीलें : ताटां ओळगवीलें आंणि वीलक्षेण देखिलें आंणि म्हणीतलें : “ऐया माझा : आतां होए म्हणे : आवो गोड असे म्हणे :” मग माहादइसें हरीखैलीं : आंणिकें वाढीलीं : ऐसीं गोसांवीं आरोगणा स्वीकरीलि : ॥८८॥

    • ९) टोकणया ग्रहीं मादणे

  • ९१ टोंकणया ग्रहीं मादणें गोसावी अनुकारू करूं लागलें : “आवो सनिवारीं न्हावें म्हणे : आवो बुधवारीं न्हावें म्हणे :” ऐसे अनुकारु करूं लागले : ते आइकोनि माहादाइसीं वाटी भरि तेला हातीं घेतली : न पुसत गोसावियांचां श्रीमुगुटी तेल ओळगवीलें : आणि गोसांवीं कोपु नटले : ते वाटी श्रीचरणें हाणीतली : आंणि गोसांवीं बीजें केले : मागेंचि माहादाइसें नीगाली : तवं दादोस माहाद्वारीं भेटले : दादोसीं म्हणीतलें : “रुपै ! रुपै ! ओखटें करीत असा : जेणें श्रीप्रभुची प्रवृत्ति भंगे तें तें करीत असा : जेणें नरका जा तें तें करीत असा : एथ काइसया आहा ? नरका जावेया ?” माहादाइसीं म्हणीतलें : “दादो ! तुम्हीं उगे असा हो : फाटका पाओ सों नका : गोसांवी उदास जाले तें एक : आणि तुम्हीं कोपतां तें दुसरें : ”आंणि गोसांवीं तैसेचि टोंकणयां (चेया) आवारासि बीजें केले : ओसरीएवरि रीगतां डावेया हाता (भीतीसीं) उकड आसन : तवं मागिलाकडौनि माहादइसें गेली : तीयें उजीवेनि सीहाडेनीसी दीनवदनें उभीं राहीली असेति : गोसावियांचीए श्रीमुगुटींचे तेल नीतळों लागलें : तें श्रीकरें भीतीसी पुसीति : तवं माहादाइसें वीनवीलें : “जी जी ! आमचे गोसांवी म्हणीतलें करीति : वीनती स्वीकरीति : सेवा करूं देति : जी जी ! तरि मीं पुसुं जी ?” “आवो ए ये नां म्हणे : पुसि नां म्हणे :” माहादाइसें गेली : कासवटा दीधला : श्रीमुगुट चोपडुं लागलीं : तवं टोंकणयाची ब्राह्मणि सोनाइसें : तींयें वाटीभरि तेल घेउनि आलीं : मग जाडी घेउनि आलीं : तवं माहादाइसीं म्हणीतलें : “जी जी ! आसन रचुं दीजो जी !” मग महादाइसीं आसन रचीलें : मग सम-स्तळ आसन जालें : मग सोनाइसीं म्हणीतलें : “आइ ! डेराभरि पाणिया तापतु असे : आइती चोखनी असे :” तें तेल माहादाइसीं श्रीमुगुटीं घातलें : डोळां घातलें : कानीं घातलें : मग गोसांवी एकु एकु श्रीकरु दोनि दोनि वेळ देति : एकु श्रीचरणु दोनि दोनि वेळ देति : तवं माहादाइसां सुखानंदु होए : मग गोसांवीं म्हणीतलें : “आवो मेली जाए : आतां होय म्हणे :” ऐसी मर्दना जाली : मग माहादाइसीं भटातें म्हणीतलें : “नागदेवा ! जाए : तेया डेरेयाखाली जाळु घाली :” भट गेले : तेया डेरेयाखालि जाळु घालीत असेति : माहादाइसीं मर्दना दीधली : डेराभरि पांणीं सोनाइसीं ठेवीलें होतें : तेणें मादनें होत असें : भटातें म्हणीतलें : “नागदेया ! तो डेरा घेउनि ए हो :” मग भट गेले : तवं तो डेरा काठासहीत तापला असे : भटाचेनि तो डेरा नाणवेचि : मग मागुते आले : “रुपै : तो डेरा थोरु तापला असे :” मग माहादाइसें गेली : चुलीचा उबाळा काढीला : तो वीझवीला : चुंभळि चीखलें माखीली : तळहातावरि घेउनि आली : तीए डेराचें पाणीं : ऐसे दोहीं डेरीयाचेनि पाणींयें गाडु ढाळीला : ऐसें मादनें जालें : मग माहादाइसीं पीवळीए पाजणीचा साउला ओळगवीला : मग सोनाइसाचीं लेंकरुवें जीति नां : तें मृतवांझ : तयासि गोसावीं पडदणीं दीधली : गोसांवीं म्हणीतलें : “आवो : घे घे नां म्हणे :” म्हणौनि डावीए श्रीचरणीचेनि आंगुठेनि तीएवरि घातली : मग माहादाइसें मागों बैसलीं : “पडदणीं दे ओ :” तीया म्हणीतलें : “हें काइ वो जालें : आमचां राउळीं मज पडदणीं दीधली : मीं काइसेया देइंन :” तीयेसि तीयेचि दीसीं गर्भसंभुति जाली : नवा मासा तो पुत्र जाला : तेथ गोसावीं बीजें केलें : नाळातें गोसावी म्हणति : “आवो नाळ होए म्हणे : आवो सापु होए म्हणे : आवो नाळु होये म्हणे :” मग नाळ खांडीलें : मग गोसांवीं पेवावरि बीजें केलें : “आवो सात खांडीया वेचावीया म्हणे : पांच खांडीया वेचावीया म्हणे : : रीधौरें जेववावें म्हणे : बेलौरें जेववावें म्हणे : चौ गाउ जेववावा म्हणे : नेजवावा म्हणे : आवो जेववावाचि म्हणे :” म्हणौनि पेव श्रीचरणेंकरुनि दाखविलें : मग तीहीं बारसें केलें : रीधौरें बेलौरें जेववीलें : चौ गाउ जेववीला : आउघा लोकु जेवीला : परि सोनाइसें न जेवीतीचि : घरीचां म्हणीतलें : “आवो तु जेवीसि नां कां ? तीया म्हणीतलें : “राउळ एति : मग जेविन तयाचीए पांतीसि:” तीही म्हणीतलें : “ आवो ! राउळ वेडे : राउळ पीसे : ते काइ एति ? तु जेवि : लेंकरुवासि दुध कडु होइल” : “नां : नेजवी :” तवं गोसांवी बीजें केलें : मग गोसावियांसी सीरा ताट केलें : आरोगणा : सोनाइसामुक्षकरुनि एरे आवघीं गोसावियांचीए पांतीसि जेवीली : तो पुत्र सतायु जाला : तयासि जरु ताओ नांहीं : खोकला पडसा नांहीं : कोणे व्याधीचा उपद्रो नाहीं : मग गोसावीं बीजें केलेयाउपरि आपुलीए अनुभवाचीया लीळा भगतीजनापुढें सांघति : ॥९१॥

    • १०) गुळ दरसनि म्हाइंभटा भेटि

  • ९७ गुळुदरसनि म्हांइंभटां भेटि म्हांइंभट सराळाहुनि भावैचां चतुर्दसीं आले : पाटांउ दुटी पांगुरले होते : ते एका ब्राह्मणासि दीधली : तयाची घोंगडी आपण घेतली : ते सतरा दीसां रीधपुरासि आले : ते वाटे सीराळां पासवडीची पाटी आठें दांमीं वीकीली : तेथ एका दांमीयांची पाटी घेतली : सांता दांमांचा गुळु घेतला : मग रीधपुरासि आले : तवं श्रीप्रभुगोसांवियांसि तीकोपाध्याचीए आवारीं अवस्थान असे : आंगणीं वेढे करीत असेति : तेवीचि म्हणेति : “आवो मेला जाए : म्हांइंया यइल म्हणे : गुळु आंणील म्हणे : गुळु खावा म्हणे : ना खावा म्हणे : आवो खावाचि म्हणे :” तैसेचि पटीसाळेवरि गोसांवीं बीजें केलें : वोटेयावरि आसन जालें : आणि श्रीदोंदाकडे अवलोकुं लागले : श्रीकरीचीया आंगुळीयाकरुनि रेघा काढीति : ऐसे अनुकार करूं लागले : तवं म्हांइंभट दक्षीणीलेनि दारवठेनि आले : दावठेया आंतु रीगतां मांगु भेटला : तयांतें म्हांइंभटीं पुसीलें : “श्रीप्रभु कवणी ठांइं राज्य करीत असेति ?” मातांगें म्हणीतलें : “हें नव्हे : आतांचि एथ खेळत होते : ऐसेंचि तीकोपाध्यांचेया आवारासि बीजें केलें :” तो गावांबाहीरि नीगाला : म्हांइंभट गावांतु गेले : तो पाठिमोरा जाला : आंणि म्हांइंभटी साष्टांग दंडवत घातलें : मग तीकोपाध्यांचेया आवारासि आले : श्रीप्रभुगोसांवियांसि दर्सन जालें : गुळाची पांटी दरिसना केली : सोनेसळा फुटा ओळगवीला : मग दंडवत घातलें : श्रीचरणां लागले : मग गोसांवी म्हणीतलें : “मेला जाए : आतां होये म्हणे :” म्हणौनि श्रीकरीं गुळु घेतला : श्रीमुखीं घातला : खावों लागले : “आवो मेला जाए : गोड आहे म्हणे :” म्हणौनि दुसरेनि श्रीकरें भेली श्रीमुखीं घातली : आणि आरोगुं लागले : ते श्रीमुखुनि ओघळु नीगेति : तें खाड माखे : दोंदावरि गळे : ऐसा गुळु आरोगीला : म्हांइंभटासि प्रसादु दीधला : तवं आबाइसें जेवीतें होतीं : तीए लवकरि आंचवली : बाहीरि आली : “ जी जी ! पुरे कीजो : पोट दुखैल :” म्हणौनि आबाइसीं दुरडी भीतरि नेली : मग उदक आणिलें : गोसावियांसि गुळुळा करू घातला : श्रीदोंद प्रमार्जीति : मग आबाइसीं पुसिलें : “भटो ! कवणी ठांउंनि एत असा ?” म्हांइंभटीं म्हणीतलें : “मी सरालां होउनि आलां :” मग आबाइसीं म्हणीतलें “आमचे गोसांवी कवनीं ठाइं राज्य करीत असेति ?” मग आबैसीं पाये धुतले : भीतरि ठाओ केला : “भटो ! या : भीतरि ठाओ केला असे :” मग म्हांइंभट जेवीले : आंचवलें : मग म्हांइभटीं आबाइसासि दंडवत घातलें : “मज चुलीची राख पुंजा सांडुं देयावा :” मग म्हांइंभट गोसावियांचें सेवा-दास्य करुं लागले : ॥९७॥

    • ११) काळवीहीरीचा काटा मोडणें

  • १०० कोळवीहीरीचा कांटा मोडणें एकु दीं गोसावियांसि ताट जालें : ताटीं उपवीष्ट जाले : आणि तैसेचि ताटावरुनि बीजें केलें : चरणचार्य धांव घेतली : मागीलाकडुनि भगत-जनीं धांवं घेतली : कोळवीहीरीचेया वेदीदारासमीप बाबुळ होती : तेथ गोसांवीं बीजें केलें : तीचा कांटा श्रीकरें मोडीला : मग म्हणीतलें “मेला जाए : हा तो नव्हे म्हणे : आवो होयचि म्हणें :” म्हणौनि मोडौनि टाकीला : आंणि परतले : मग गोसांवी बीजें केलें : मग एउनि आरोगणा केली : ॥१००॥

    • १२) तथा नारीएळें दामुर्तांते हाणणे

  • १०९ तथा नारीएळें दामुर्तातें हाणणें : भट म्हांइंभट संबोखु मग गोसांवीं दामुर्तासि पाठवणीं दीधली : परि तो उगाचि बैसला होता : मग गोसांवीं म्हणीतलें : “आवो मेला जाए : जाए नां म्हणे :” एकु वेळ पाठवणी दीधली : दोनि वेळ दीधली : तीसरेनी दीधली : परि तो न वचेचि : तवं भेटीसि नारीएळें आलीं होतीं : ते गोसाविया पुढें असेति : गोसांवीं एकु नारीएळ श्रीकरीं घेतलें : मग गोसावी म्हणीतले : “मेला जाए : उठि उठि नां म्हणे :” म्हणौनि कोपु नटले : आणि नारीएळें दामुर्तातें हांणितलें : तो उठीला : घरासि गेला : भक्तिजनीं म्हणीतलें : “तेथ तरि तें केलें : आणि एथही ऐसें गोसांवी काइ करूं लागलें !” म्हणौनि भेवों लागले : मग गोसांवियांसि मर्दना मादणें जालें : पुजा आरोगणा जाली : मग पहुडु जाला : मग भट म्हांइंभट पडीताळावया गेले : भट म्हांइंभट आले म्हणौनि दामुर्त उठीला : बैसावया घातलें :मग बैसले : मग भटीं म्हणीतलें : “दामुर्तो ! तुमचे कपाळ कैसें असे ?” तीही म्हणीतलें : “बरवें असे : हें नव्हे : राउळ आतांचि पडीताळावया आले होते : श्रीकरें कपाळ स्पर्सीलें : मग म्हणीतलें : ‘आवो मेला जाए : दुखवला म्हणे :’ ते माझें दुख नीरपौनी गोसांवीं फेडिलें : ‘आवो दुखविलासि म्हणे :’ म्हणौनि श्रीकरें माथा स्पर्सीला :” मग भटां म्हांइंभटा थोर सुख जालें : मग भटीं म्हणीतलें : “गोसावियांसि मर्दना मादणें जालें : पुजा आरोगणा पहुडु जाला : संदि पडीली : मग आम्हीं आलों :” मग दामुर्तें म्हणीतलें : “हा भटो ! आणीकी आर्वाचीकां धुरांचा ठांइं पाठवणीं दीधलेयां अवसि बैसों न ये : ऐसें जरि तयांचा ठाइं तरि गोसांवीं पाठवणीं देतां मीं न वचें : एकु वेळु पाठवणीं दीधली : दोनि वेळ दीधली : तीसरेनीहीं दीधली : परि मी न वचेंचि : मग गोसांवीं सीक लावीली : माझी खोडी सांडवीली :” ॥१०९॥

  • १३) सर्पदृष्ट माहादाइ रक्षणें

  • १३१ सर्प-दृ(द)ष्ट माहादाइ रक्षणें एकु दी माहांदाइसें पाहारा रात्रीं उठीली : तांब कळस घेतले : मग गेलीं : तांब कळस भरीले : कावडीसि घातले : घेउनि नीगाली : तवं पांन लागलें तैसीचि कावडि घेउनि आलीं : रायांगणीं उतरीली : आणि मुर्छना एउनि पडीलीं : ते समइं गोसावियांसि माचेयावरि आसन : तवं आबाइसें बोबाइली : “जी जी ! देवकी दासि सर्पें खादली जी !” तैसेंचि गोसांवीं रायांगणासि बीजे केलें : दृष्टि अवलोकीली : वीख नीरपौनि काढलें : मग उठीलीं : गोसांवियांसि दंडवत घातलें : श्रीचरणां लागलीं : मग गोसांवीं म्हणीतलें : “आवो मेली जाए : माहादी सापें खादली म्हणें : की माहादीया सापु खादला म्हणे ?” ॥१३१॥

    • १४) लेंकी जावइ एतें म्हणणें

  • १५६ लेंकी जावाइ एतें म्ह(ण)णें एकु दी गोसांवी रायांगणीं वेढे करीत असेति : गोसांवीं म्हणीतलें : आवो मेला जाए : (लेंकि जावाइ) एइल म्हणे : नैये म्हणे : आवो येइलचि म्हणे : लेंकि उटावी म्हणे : (न्हाणावी म्हणे) : चीर चोळी घेयावी म्हणे : (बरवें जेउं घालावें म्हणे : जावाइयासि पासवडी घेयावी म्हणे) :” तव सांवळापुरीची एकि (एका) ब्राह्मणाची सुन : तीसि माय बापु नांहीं : भावु बहीणि नाहीं : तीसि थोरु सासुरेवासु करीति : एकु दी तीचेनि भ्रातारे मारीली : आणि सवी दीधली : “ जाय नीदैवांचे नेदैवे! ” ते रोसे नीगाली : तीचेनि वरैतें पुसीलें : “तुं कें नीगालीसी? ” तीया म्हणीतलें : “ मी आपुलेया माहीयरा जाति असे : ” “के असे तुझें माहीयर? ” तीया म्हणीतलें : “ हे नव्हे रीधपुरीं असे :” “तेणें म्हणीतलें : “तेथ कोण असे? ” “ना राउळ आहाति :” म्हणौनि नीगाली : तीयें मागे तोहि नीगाला : म्हांइंभटी म्हणीतलें : “ आजि गोसावियांची लेंकि कवणी पां यइल? ” तवं ते आली : गोसावीं लेकीसि खेवं दीधले : ते गोसांवियांचा गळा धरुनि रडु(डो) लागली : (मग जावाइयें खेवं दीधलें:) मग आबाइसी तीसि उटीले : न्हाणीलें : जेउ घातलें : पांच दीस होती : मग म्हांइंभटी चीरचोळी घेतली : जावाइयासि पासवडी घेतली : मग ते गेली : मग एकु दीसीं दुरडीभरि धीडरीं (धिरडीं) वेढी (वाढे) भरि तुप घेउनि आली: गोसांवियापुंढें ठेवीली : गोसांवियांसि माचेयावरि आसन असे: गोसांवी धीडरीं मोडीति : आणि वेढि (वेढैये) आंतु चुभळीति : आरोगीति : मग आबैसीं दुरडी परती काढीली : गोसांवी म्हणीतलें : “ अवो मेली जाए : गोड आहे म्हणे: आवो काह्य नेतासि? ” आबाइसी म्हणीतलें : “ जी जी! आतां पुरे कीजो जी ! मग श्रीकरें श्रीमुख प्रक्षाळीति : मग गुळळा वीडा जाला : ते आपुलेया गावांसि आली (गेली) :॥१५६॥

    • १५) साधा दाढ, नख, फोडी केसकळाप- प्रसाद दान

  • १७५ साधां दाढ-प्रसादु दांन गोसांवियांसि माचेयावरि आसन असे : गोसांवियांची दाढ हालत होती : मग गोसांवीं श्रीमुखीं आंगुळि घातली : आंणि दाढ काढीली : जवळि साधें बैसलीं होती : गोसांवीं म्हणीतलें : “आवो मेली जाय : घे घे घे नां म्हणे :” म्हणौनि साधासीं दीधली : ॥१७५॥
  • १७६ साधां नख-प्रसाद दांन एकु दी गोसांवियांसि माचेयावरि आसन असे : गोसांवीं डावीय श्रीकरीचें नख उजवेंनि श्रीकरें मोडीलें : मग गोसांवीं म्हणीतलें : “ आवो मेली जाय : घे घे घे नां म्हणे :” म्हणौनि साधासीं दीधलें : तें साधीं पाटावाचां चीरकुटीं गुंडुनि बांधलें : मग गळां बांधलें : ॥१७६॥
  • १७७ साधां फोडि-प्रसाद दांन माहाद्वारीचें कवाड लांपट : मग तें धरावयाचें साधांसि उलींग : गोसांवी म्हणेति : “आवो डोळो डोळो :” ऐसें म्हणेति : आंणि दांडी नीगे : तवं साधें कवाड टींबकारीति : (धरिती) : आणि गोसांवी श्रीमुखींचें तांबोळ तयावरि फुकुनि घालीति : प्रसादु आवघीं भक्तजनें घेति : एकु दीसीं साधीं ऐसा मनोर्थु केला : “मी पापीनि प्रसादा चुकत असे : अवघें भगतजन प्रसादु घेत असेति :” तो मनोर्थु जाणौनि गोसांवी एकी फोडी गाली धरीली होती : मग माहाद्वारासि गोसांवी बीजें केले : आणि श्रीकरेंकरूनि मुखीची फोडि काढीली : मग गोसांवी म्हणीतलें : “आवो मेली जाय :” म्हणौनि साधासी दीधली : एकी वासना : उजवीय श्रीकरीची आंगुठां धरीली होती : ते दीधली : ॥ १७७॥
