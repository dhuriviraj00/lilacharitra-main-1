पूर्वीं

सृष्टीमध्यें कव्हणें करणें मूंचिजेना एकें परमेश्वरेंवीण: तोचि परमेश्वरु निर्वेवो निराकारु असे: परि कृपावशें सावेवो साकारु होए: अवतरे: आपुलें सन्निधान दे: तें दास्य मोचक: ।छ।

अन्यव्यावृत्ती

कर्मभूमीचिया देवता होति : परि परमेश्वरु नव्हेति ।१।

अष्टौदेवयोनि देवता होति : परि परमेश्वरु नव्हेति ।२।

अंतराळीचे गंधर्व देवता होति : परि परमेश्वरु नव्हेति ।३।

स्वर्गीचे इंद्रचंद्रादिक देवता होति : परि परमेश्वरु नव्हेति ।४।

कैलास वैकुंठीचे हरिहर ब्रम्हादिक देवता होति : परि परमेश्वरु नव्हेति ।५।

क्षीराब्धीचा शेषशायी देवता होए : परि परमेश्वरु नव्हे ।६।

अष्ट भैरव देवता होति : परि परमेश्वरु नव्हेति ।७।

विश्वरुप देवता होए : परि परमेश्वरु नव्हे ।८।

चैतन्य देवता होए : परि परमेश्वरु नव्हे ।९।

चैतन्यापर परमेश्वरु : तो देवो श्रीचक्रधरु : सकळ जीवांतें रक्षिता हो : उद्धरिता हो ।१०।

युगधर्म

बत्तीस लक्ष कृतयुग ।१।

कृतयुगीं आत्मोपास्ति हा धर्मु ।२।

भक्ती अधर्मु ।३।

धर्मातव चैतन्यप्राप्ति फळ ।४।

अधर्मातव खालीली युगीं जन्म ।५।

सोळा लक्ष त्रेत ।६।

त्रेतीं भक्ती हा धर्मु ।७।

यागु अधर्मु ।८।

धर्मातव कैलास वैकुंठप्राप्ति फळ ।९।

अधर्मातव खालीली युगीं जन्म ।१०।

आठ लक्ष द्वापर ।११।

द्वापरीं याग हा धर्मु ।१२।

तीर्थ : क्षेत्र : व्रत : दान : अधर्मु ।१३।

धर्मातव स्वर्गप्राप्ति फळ ।१४।

अधर्मातव खालीली युगीं जन्म ।१५।

च्यारि लक्ष कलियुग ।१६।

कलियुगीं तीर्थ : क्षेत्र : व्रत : दान : हा धर्मु ।१६।

हिंसा अधर्मु ।१८।

धर्मातव अंतराळप्राप्ति फळ ।१९।

अधर्मातव नरक ।२०।

उत्तमा धर्मातव अंतराळप्राप्ति फळ ।२१।

मध्यमा धर्मातव अष्टौदेवयोनिप्राप्ति फळ ।२२।

कनीयसा धर्मातव अधोनिकप्राप्ति फळ ।२३।

कृतयुगीं लक्षभरि आयुष्य ।२४।

त्रेतीं दाहा सहस्त्र ।२५।

द्वापरीं सहस्त्र ।२६।

कलियुगीं शत ।२७।

या चहुयुगांचां धर्मीं परमेश्वरु नेणिजे : न पविजे : परमेश्वरें परमेश्वरु जाणिजे : परमेश्वरें परमेश्वरु पाविजे ।२८।

विद्यामार्ग

एकि विद्या ।१।

दोनि विद्या ।२।

आठि विद्या ।३।

सवावो लक्ष नव कोडी विद्या ।४।

बाहात्तर कोडी विद्या ।५।

एकि विद्या ते ब्रह्मविद्या ।६।

द्विधा शांभव ।७।

अष्टधा शाक्तेय ।८।

सवावो लक्ष नव कोडी महाणव ।९।

बाहात्तर कोडी आणव ।१०।

विषयाधार वेधु ।११।

इंद्रियाधार वेधु ।१२।

अंत:करणाधार वेधु ।१३।

देहाधार वेधु ।१४।

विदेहा धार वेधु ।१५।

केवळाधार वेधु ।१६।

विषयवेधें विषयांची प्राप्ति ।१७।

इंद्रियवेधें इंद्रियांची प्राप्ति ।१८।

अंत:करणवेधें अंत:करणाची प्राप्ति ।१९।

देहवेधें देहाची प्राप्ति ।२०।

विदेहवेधें विदेहाची प्राप्ति ।२१।

केवळवेधें कैवल्याची प्राप्ति ।२२।

संहार

परमेश्वरप्रवृत्ती माया सृष्टिसंहारू करी ।१।

विखा इंद्रियीं विलया जाए ।३।

इंद्रिय अंत:करणीं विलया जाए ।४।

अंत:करण देहीं विलया जाए ।४।

देह विदेहीं विलया जाए ।५।

विदेह ते नित्य ।६।

कार्य अविद्या कारण अविद्येचां ठाइं विलया जाए ।७।

कार्य कारणीं विलया जाए ।८।

एकु नित्यवर्गु : एकु अनित्यवर्गु ।९।

नित्यवर्गी जीव : देवता : परमेश्वरु : उरेती ।१०।

अनित्यवर्गी अनादि अविद्या : कर्मलेप : उरेती ।११।

अनादिअविद्याकर्मलेपयुक्त जीव मायास्वरुपीं असति ।१२।

देवतास्वरुपीं स्थिता होति ।१३।

यापरी प्रपंचु संहरे ।१४।

माया परमेश्वराची प्रवृत्ति वोळगे ।१५।

संसरण

मायास्वरुपीं एकदेशीं तम असे ।१।

तमोनिमग्न जीव मायास्वरुपीं असति ।२।

परमेश्वरप्रवृत्ती माया तमोनिग्नां जीवां चैतन्यमहं ऐसें उपजवी ।३।

माया जीवांतें विकरे : संचरे : विये: प्रसवे : हालवी : डोचली : उगळी : लोटी ।४।

परमेश्वरप्रवृत्ती माया व्यापारवशें अविद्यात्मक जीव चैतन्यम् अहं ऐसे घेउनि उठीति ।५।

चैतन्यम् अहं जीवां जन्म बोलिजे : अन्यथाज्ञान बोलिजे : आद्यमळ बोलिजे ।६।

मायासृष्टि जीव : जीवसृष्टि प्रपंचु ।७।

मायाकृत जीव : जीवकृत प्रपंचु ।८।

मायाकार्य जीव : जीवकार्य प्रपंचु ।९।

यापरी माया जीवांतें जन्मौनि आपण उदासीन होए : परि सूत्रातें न संडी ।१०।

जीवु मायेसि अधीनु होए : का प्रपंचासि अधीनु होए : मध्यें असणेंचि नाहीं ।११।

जन्मलासांता जीवु क्षणपर्यंत शुद्ध असें मग मैळे ।१२।

चैतन्य मृत्तिकापर्यंत मैळे : चैतन्यासवें जीवु मृत्तिकापर्यंत मैळे ।१३।

चैतन्य चैतन्यपर्यंत उजळे : चैतन्यासवें जीवु चैतन्यपर्यंत उजळे ।१४।

आनाचां करणा तेयासि अभिमानु ।१५।

मग जीवु प्रपंचातें रची ।१६।

कां रची : जे अधमति : अधगति : अधरति म्हणौनि रची ।१७।

काइसेनि रची : तै तया सामर्थ्य आति ।१८।

मग जीवु नाहीं तेंचि एक गंधर्वनगराकार विश्व रची : तेयामाजी : रीगे : तादात्मकु होए : विश्वम् अहं ऐसे बूझे ।१९।

मग जीवु नाहीं तेची एक गंधर्वनगराकार अष्टधा प्रकृति रची : तियेमाजी रीगे : तादात्मकु होए : प्रकृतिरहं ऐसे बूझे ।२०।

मग जीवु नाहीं तेची एक गंधर्वनगराकार अनेकधा विकृति रची : तियेमाजी रीगे : तादात्मकु होए : विकृतिरहं ऐसे बूझे ।२१।

मग जीवु नाहीं तेची एक गंधर्वनगराकार अनेकधा विकृतिविकृति रची : तियेमाजी रीगे : तादात्मकु होए : विकृतिविकृतिरहं ऐसे बूझे ।२२।

यापरी मनुष्यपर्यंत मैळे ।२३।

मग जीव चतुर्विधा कर्मफळां जाति ।२४।

स्वर्ग : नरक : कर्मभूमि : मोक्ष यातें चतुर्विध कर्मफळ बोलिजे ।२५।

यापरी मागौती जैसी असिली तैसी सृष्टि रची ।२६।

माया परमेश्वराची प्रवृत्ति वोळगे ।२७।

असतिपरी

विकारविकल्पशून्यें ।१।

स्वभावमात्रें ।२।

निरालंबी ।३।

देवतास्मरणें ।४।

जो जन्म क्षेपी ।५।

तेयासि परमेश्वरु पुनरपि आपुला संबंधु देती ।६।

महावाक्य

जीवप्रपंचव्यतिरिक्त सच्चिदानंदस्वरुप परमेश्वरु एकु आति : तो सर्वशक्तियुक्त ।१।

नित्य : केवळ : अनादिअविद्यायुक्त वस्तु एकु आति ते जीवशब्दें बोलिजे ।२।

विश्वरुपाचिये ठाउनि मृत्तिकापर्यंत प्रपंचु बोलिजे ।३।

व्यतिरिक्त म्हणजे वेगळा ।४।

सत् शब्दें ब्रह्म बोलिजे : चित् शब्दें माया बोलिजे : आनंद शब्दें ईश्वरु बोलिजे : ब्रह्म : माया : ईश्वरु ऐसा त्र्यंश परमेश्वरु एकु आति ।५।

शक्ति शब्दें सकळही देवता : सकळही गुण : सकळही धर्म : ज्ञान : विज्ञान : हे सर्व शक्तिशब्दें बोलिजेति ।६।

युक्त म्हणिजे एतत्सहित ।७।

सौंदर्य : लावण्य : औदार्य : सौभाग्य : दया : मया : कृपा : करुणा : एवमादिक अनंत गुण बोलिजेति ।८।

नित्यत्व : व्यापकत्व : सर्वात्मकत्व : सर्वातीतत्व : सर्वकर्तृत्व : सर्वसाक्षित्व : एवमादिक अनंत धर्म बोलिजेति ।९।

निर्वचन

आनंदशब्दें ईश्वरु बोलिजें ।१।

आनंद तो ।२।

ब्रह्म तो ।३।

अच्युत तो ।४।

अविक्रियु तो ।५।

अव्यक्तु तो ।६।

अमूर्त तो ।७।

शुद्ध तो ।८।

बुद्ध तो ।९।

नित्यमुक्त तो ।१०।

निरभिमानु तो ।११।

पर तो ।१२।

परसाक्षित्व ऐसाही धर्मु तेथ नाही ।१३।

केवळ तो ।१४।

केवळींचि नित्यसंबंधे माया असे ।१५।

माया जेथ संबंधातें पातली असे तेथचि न पवे ।१६।

ऐसीचि वस्तु ते ।१७।

ऐश्वर्यादिकरौनि सकळही धर्म मायासंबंधेचि तेथ ।१८।

सर्वात्मकु तो ।१९।

सर्वातीत तो ।२०।

सर्वकर्ता तो ।२१।

सर्वसाक्षी तो ।२२।

ऐसा परमेश्वरु एकु आति ।२३।

उद्धरण

सृष्टीमध्यें परमेश्वरु अवतरेति ।१।

मनुष्यवेषधारीए होति ।२।

मायावेषधारीए होति ।३।

मायापुरधारीए होति ।४।

मायानिर्मितं पुरं मायापुरम् ।५।

माया घटाकारें आकारे ओते ।६।

नख : चर्म : रोम : आदिकरौनि शुद्ध चैतन्यचि ते ।७।

मायापुरीं मायेसि अभिमानु ।८।

ब्रह्माण्डीं शक्तिअधीनु शक्तिव्यापारु : मायापुरीं ईश्वराधीनु शक्तिव्यापारु ।९।

मायापुरीं इतरशक्तितुल्य माया ।१०।

ईश्वरीं संपुर्णा शक्ति : जीवीं अंशत: शक्ति ।११।

ऐसेया मायापुरातें स्विकरौनि कर्मभुमि स्थितां मनुष्यदेहयुक्तां जीवां ज्ञानपर्यंत परमेश्वरु आपुला संबंधु देति ।१२।

एकां प्रेम संचरीति ।१३।

प्रेम बोलिजे : भक्ति बोलिजे ।१४।

प्रेमसंचारमात्रें विकारु : विकल्पु : स्वभावो इये तीनि उच्छेदौनि जाती ।१५।

भक्ताचे सकळही सन्निधीनीं चरितार्थ होए ।१६।

भक्तासि श्रीमुर्तिवरी साध्य साधन ऐसी प्रतीति : प्राप्य प्रापक बोधु ।१७।

साध्य साधनाकारें आकारलें : करचरणवंत ब्रह्म हें ऐसी भक्तासि प्रतीति ।१८।

भक्त वियोगीं नुरे ।१९।

भक्ता नित्य संबंधु ।२०।

एकां जीवां प्रति परागिराकरौनि परमेश्वरु आपुले अस्तित्व साधीति ।२१।

मग एक अनुसरेति : एक ननुसरेति ।२२।

ननुसरेति ते प्रमादिये बोलिजेति : तेयां सृष्टि शून्य होए ।२३।

अनुसरलेयाचे आगंतुक कुंठे : अनारब्ध सुंके : प्रारब्धां भोगें क्षयो ।२४।

अनुसरलेयासि कर्मसवा भोगु नाहीं : दैवसवा भोगु ।२५।

अनुसरलेयासि देवत्व : तिर्यंचत्व होणें नाही ।२६।

ज्ञानियासि श्रीमूर्तिवरी साधन प्रतीति प्रापकु बोधु ।२७।

कैवल्यदाता हा : मोक्षदाता हा : ऐसी ज्ञानियासि प्रतीति ।२८।

अनुसरौनि सन्निधान करी : यापरी योग्य होए : तेयासि परमेश्वरु सत्ताज्ञान दान करीति ।२९।

कव्हणी एकीं अवसरीं सकळही मळांचे परिसारण करौनि परमेश्वरु जीवीं आपुली सत्ताशक्ति संचरीति : मग मागौते मळ घालीति ।३०।

सत्ता बोलिजे : सूचा बोलिजे : अव्यभिचार्य ज्ञान बोलिजे ।३१।

सत्ता ते विदित ।३२।

सत्ताज्ञान तेंचि अपरोक्ष ज्ञान ।३३।

सत्ताज्ञानी अशेष कर्म जाणे : कर्मक्षाळणी उपाओ जाणे : एतें विघ्न जाणे : विघ्नोपशमु जाणे ।३४।

सन्निधानीं क्षाळी कां असन्निधानीं क्षाळी ।३५।

सन्निधानीं तोकडेनि बहुत क्षाळी : असन्निधानीं जें जेतुलेनि क्षाळावें तें तेतुलेनि क्षाळी ।३६।

सत्ताज्ञानीं विकारसृष्टि : कर्मसृष्टि : अधीना होति : दैव विघ्न ।३७।

ब्रह्मादिक आपुुलेया पदा नेयावेया एति : परि तो तयांची वासही न पाहे ।३८।

अष्ट भैरव हालवीति : डंडळवीति : परि तो जैसा काहीं तैसाची ।३९।

ब्रह्मतुल्य आनंद विघ्नत्वें प्रकटति ।४०।

विद्या : अविद्या : पुरुषु : यां तीहीतें विद्या प्रकाशी : मग तो विद्येअधिनु होए ।४१।

यापरी योग्य होए तेयासी परमेश्वरु सामान्यज्ञान दान करीति ।४२।

सामान्यज्ञानी विकारसृष्टिकर्मसृष्टि वेगळा होए : दैवसृष्टि अधीना होति ।४३।

सत्ता सामान्यज्ञान मिळौनि अशेष कर्म क्षाळण करी : अशेष मळ क्षाळण करी ।४४।

तेणें लाविले तेेंं तो फेडी : तेणें नाहीं लाविलें तें तयाचेनि न फिटे ।४५।

यत्क्रियते तत्कर्म : म्हणौनि कर्म : कर्म शब्दें कर्म बोलिजे : कर्म शब्दें मळही बोलिजेति ।४६।

यापरी योग्य होए तेयासि परमेश्वरु विशेषज्ञान दान करीति ।४७।

विशेषज्ञानी दैवसृष्टि वेगळा होए : सकळाही शक्ति अधीना होति : अपरोक्षा होति ।४८।

माया कोपौनि उदासीन होए ।४९।

मायाकृपाव्यतिरिक्त परमेश्वरु तेयासी सकळाही शक्ति दान करीति ।५०।

मग तो सकळांसिही प्रमाणद्रव्य होए ।५१।

आद्यमळयुक्त चरमदेहस्थित कृपास्पदीभूत होउनि असे ।५२।

मग परमेश्वरु तेयासि कृपा करीति ।५३।

मग परमेश्वरु आपुलिया कृपाशक्तिकरुनि तेयाची अनादि अविद्या छेद करीति : मूळ अविद्या छेद करीति : अज्ञान छेद करीति ।५४।

अज्ञान छेदेंकरुनि अन्यथाज्ञानत्रुटि होए : आद्यमळत्रुटि होए : जीवत्वत्रुटि होए ।५५।

तळिचील हाणितलेयां वरिचील पडे : उतरंडी गडबडी ।५६।

यापरी परमेश्वरु तेयातें माया पारंगत करीति ।५७।

साध्यांचिया प्राप्ति साधन निवर्ते ।५८।

परमेश्वरु भक्तासि आपुली अनुभूति रति देति ।५९।

भक्तासि आपणपें समोर देति ।६०।

ज्ञानियासि ब्रह्मीभूत करीति ।६१।

ज्ञानियासि पाठीसि घालीति ।६२।

ज्ञानियासि मोक्ष देति ।६३।

ज्ञानिया ब्रह्मीं मिसळे : बुडी दे ।६४।

भक्त भेदें अनुभवी : कीं अभेदें अनुभवी : ऐसा विकल्प तेथ करों न ए ।६५।

यापरी परमेश्वरु श्रीचक्रधरु जीवांतें उद्धरीती।६६।

आचार

स्वदेशसंबंधु त्याज्य : स्वग्रामसंबंधु त्याज्य : संबंधियांचा संबंधु तो विशेषता त्याज्य ।१।

संबंधीं अधिकारु नाहीं ।२।

संबंधीं विकारु : विकारीं रसु : रसीं अभिमुख : सेवा निवृत्ति ।३।

नावडत नावडत सेवी : तेंचि आवडौनि ठाके : मग सांडितांहीं न संडे ।४।

ऐसा विखो बापुडा : जो सेवितसांतां आपुुलेया मोहरा न करी ।५।

विषयसंबंधेमात्रें जीवु काइ गा जातु असे : परि सेवितसांतां तिए एकीं गडदरा घाली : जेथ काइसेनही उसंत नाहीं ।६।

पुरुषु जेतुल जेतुली विषयसेवा करी : तेतुल तेतुला धर्मीं निबळ जाए ।७।

बाइ : पाप तरि नव्हे : परि धर्मापासौनि ऐसा निर्पौनि काढी ।८।

स्त्री म्हणिजे मत्तद्रव्याचा रावो गा : आणिकें द्रव्यें सेविलेयां माजवीति : स्त्री दर्शनमात्रेंचि माजवी ।९।

चित्रीची स्त्री न पाहावी ।१०।

जियेचा एकु अवेवु देखिलेयां अलगाचिये मुनैवरि घाली : पाठीं सरणावरि तियेचें सर्वांग देखे : परि आड पालव वोडऊनि जाए : कां माजवी तें तेथ नाहीं कीं ।११।

स्त्री दर्शने माजवी : द्रव्य अस्तित्वें माजवी ।१२।

जेणें संबंधें विकारु उपजे तेयाचा असंबंधु कीजे ।१३।

विषयाचिये एकासनी न बैसावें ।१४।

विषयासाक्षी न ठाकावें ।१५।

कोमळज्ञानीं विखो विघ्न ।१६।

नरवीर आति : परि विखयवीर नाहीं ।१७।

धर्मनष्टु आति : परि विखयनष्टु नाहीं ।१८।

पुरुषु जवं जवं पळे तवं तवं विखो पाठीं लागे : पुरुषु पाठीं लागे तवं तवं विखो पळे ।१९।

तुष्टला जनु तरि विखो संपादील : रुसला तरि प्राणा घतैल : म्हणौनि उभयतां जनसंबंधु त्याज्य ।२०।

जाणतेया विखो सुखातें नेदी ।२१।

आपणेयांतें कव्हणी नेणे : आपण कव्हणातें नेणिजे : ऐसेया स्थानीं असावें ।२२।

कानडदेशां : तेलंगदेशां न वचावें : ते विषयबहळ देश : तेथ अवधूत मान्य ।२३।

महाराष्ट्रीं असावें ।२४।

मातापुरा कोल्हापुरा न वचावें : तिये साभिमानियें स्थानें : साधकांसि विघ्न करीति ।२५।

देशाचां सेेवटीं झाडातळीं जन्म क्षेपावें ।२६।

हें देखिलें तैसें आठवीजे ।२७।

नाम : लीळा : मुर्ति : चेष्टा : ।२८।

स्मरण : अनुस्मरण : सकृत स्मरण : ।२९।

तुम्हां शयनासनीं : भोजनीं : परमेश्वरुचि होआवा कीं गा ।३०।

ऐसें होए ऐसें म्हणिजे हो।३१।

एक दोनि वेळ यातें आठविजे : परि वंध्यु दिवसु जाओं नेदिजे ।३२।

अनंतु सामाये तेथ आणिकु न सामाये : आणिकु सामाये तेथ अनंतु न सामाये : कां अनंतु तो अनंतु की ।३३।

इहामुत्र फलोपभोगविरागोवैराग्यम् ।३४।

गडीं सिडीं असावें ।३५।

हाळियां पाळियां असावें ।३६।

एका झाडाची सवे न होआवी: एका स्थानाची सवे न होआवी ।३७।

भोगस्थानीं न बैसावें ।३८।

तीर्थ : देवता : पुरुषु : इये तीनि आपलवीति ।३९।

लेपाचा देउळीं न बैसावें ।४०।

लिंग : चिन्ह : मुद्रा : न होआवी ।४१।

विषयां पात्र न होआवें ।४२।

मनुष्यमात्र होउनि असावें ।४३।

जोगिया तो मसींघसीं लीतला होआवा कीं गा ।४४।

जीवासि वैराग्य मिरऊं आवडे हाही एकु विकारुचि कीं गा ।४५।

जीवु विकारा वेगळा केव्हेळांहि झालाचि नाहीं ।४६।

बाइ : वेडेयाचे विकार ते बाहीरे प्रकटेति ।४७।

बाइ : स्वभावासि नाशु नाहीं ।४८।

हातीं काडी धरीजे हाही आदिकरुनि एकु विकारुचि कीं गा ।४९।

ये दसे कायर्र्संन्यासु आति पर कारणसंन्यासु नाहीं ।५०।

रजस्तमें निरसौनि सत्वस्थितां धर्मी अधिकारु ।५१।

आचरे तेयाचा धर्मु ।५२।

निरसां : निराशां : निराश्रयां होऊनि असावें ।५३।

सुखश्रेयां वेगळेयां होआवें ।५४।

रसक्रिये वेगळेयां होआवें ।५५।

कर्म : धर्म : विधि : विखो : परित्यजौनि परमेश्वरां शरण रिगावें ।५६।

वृत्तिअधिन न होआवें : स्थितीअधिन न होआवें ।५७।

विकारु हा इंद्रियधर्मु : विकल्पु हा मनोधर्मु परित्यजुनि परमेश्वरां शरण रिगावें ।५८।

मसनीचें लाकुड तेंही वरि एका उपयोगां जाये परि आपण कव्हणाचेयाही उपयोगां न वचावें ।५९।

तुमचेनि मुंगी रांड न होआवी ।६०।

तुम्हा पडिलेयां धुळी न लागावी ।६१।

पुरुषु आपुले करणीय म्हणौनि प्रवर्ते तेयां द्रव्याचीं शतें मिळती ।६२।

एथौनि तुम्हीं ऐसेयां होआवें कीं : हा जनु तुमतें डोइये डोइये मारील : परि तुम्हीं डोइचि ओडवावी कीं ।६३।

तुम्हां मारीतां पुजीतां समानचि होआवा कीं गा ।६४।

मुक्तीची चाड तेणें मुंगिये वीरु न चिंतावें ।६५।

मुक्तीची चाड आणि तेवीचि खुंटदावें सांभाळणें ।६६।

आपण कव्हणा अधिन होआवें नाहीं : आपणेयां अधिन कव्हणातें करावें नाहीं : जीवें परमेश्वरां अधिन होआवें ।६७।

जन्मजीवित परमेश्वरां देयावें ।६८।

ऐसा असे वटोवटो करीतु : आपुले च्यारि दीस कडे घाली : मग एकु दीसु नव्हेरे म्हणौनि जावोंचि लागे ।६९।

तळमळीत जन्मातें धाडी : तेयासि परमेश्वरु आपुलें प्रेम देती ।७०।

अप्राप्तिनिमित्तकें दु:खें कर्माचीया राशि नासति ।७१।

एवं पर्यंत प्रयत्निया एकु आति जो आपुलें करणीयें करौनि देशाचां सेेवटीं झाडातळीं जन्मातें धाडी ।७२।

कांटिये एकीं तळीं सुकौनि पाडी ऐसाही एकु आति ।७३।

जीतेंनि मृत्याचां धर्मी वर्तावें ।७४।

धड तुटलेयांहीं जीवें परमेश्वरातें न सोडावे : वासना न संडावी ।७५।

खळाळीं मोकलावें ।७६।

हाडांची रांगवळी करावी नाएका ।७७।

कटंकटीययेमान होउनि असावें ।७८।

सुखसाधनें देवो न पविजे नाएका ।७९।

तेयाचे उठणेंबैसणें इतुलेंही वायां न वचे ।८०।

चातुर्वण्यं चरेद्भैक्षम् : या शास्त्रासि अनुसरीजे ।८१।

घर न निवडीत भिक्षा किजे ।८२।

वोळखितयाचेया घरा न वचावें ।८३।

तोरणमंडळाचेया घरा न वचावें ।८४।

एकिकडौनि रिगीजे : एकिकडौनि निगीजे ।८५।

नगरांतु कां पात्रांतु : नगरांतु जालें आणि पात्रांतु नव्हेचि तरि आपजवीजे ।८६।

ना तरी खुंट गिळीजे ।८७।

बाइ : तेयांसि असे तवं तुम्हासि असे ।८८।

एकी दी न मीळे दोनि दी न मीळे तिसरां दी न मिळोंचि न ये ।८९।

भिक्षा म्हणिजे पारध गा ।९०।

भिक्षा राज्य बाइ ।९१।

हां गा ऐसें अणु अणु सांडावें : ते काइ तुमचे दाइ : आपणयां लागे तेंचि मागीजे कीं ।९२।

पोरा : वेळो वेळां टीळेया जानिवेया पासि कां जासी: काइ एकु उपवासु पडिला कीं दोनी उपवास पडिलें हां गा मा नावेक तरि पांतासि कां ।९३।

बाइ : तुम्हीं नित्योपवासिये कीं ।९४।

बाइ : मिनलेयाही न सेवीजे : मग तैसीचि सवे होए ।९५।

घेइजे ऐसें कुसकरौनि येर्‍हें येरु परतें ।९६।

प्राणासि आहारु देआवा : इंद्रियासि नेदावा ।९७।

इंद्रियार्थ रसु न सेवावा ।९८।

घेइजे ऐसें मुसंब खोउनि : आंतु पोकळ असे : खालीं ओहटळ असे : जाइल परते ।९९।

भदें भरौनि ठेवीजे : मागे तेव्हेंळीं तेव्हेंळीं घालीजे : नेघे तरि म्हणीजे : खासील तरि खाये : न खासील तरि धूळि खाये : मरों लागसी तेव्हेंळीं आपसेयाचि खासी ।१००।

परिच्छेदीं प्रवर्तलेया पुरुषाचे सकळही निरोध नेणों कोणचि निर्पौनि फेडी ।१०१।

पुरुष प्रयत्नी दैवाचें साह्य ।१०२।

धर्माचिया चाडा रायाची राणी कां न ये : वरि पडतें आकाश धरिजैल ।१०३।

न भीया हो साधे हो : तुम्हांवरि एथौनि वाजत वारा धरीजैल ।१०४।

सी : बी : चूळ : चीमुटि : यां चौमध्ये न गमे तरि एकु ठावो आपजवीजे ।१०५।

त्रसों नेदीजे : त्रसला सांतां जीवु विषयां शरण जाइल ।१०६।

त्रसला सांतां देशाचेया सेवटां जाउनि सर्वाधमत्व ब्राह्मणत्व स्वीकरील ।१०७।

अल्पें अनिष्टें बहुत अनिष्ट परिहरे तरि परिहरीजे ।१०८।

पोेरें हो : तुम्हीं कांटेया अधीन होआल ।१०९।

तैसें काइ बाइ : न गमे तरि तीनि वेळ घेतले तरि काइ ।११०।

नीरसेंं विरसेंअन्नें सेवीजेति ।१११।

रेंदेया चेंदेया दरु भरीजे ।११२।

जेव्हढी पीसवी किजे तेव्हढी होए ।११३।

धाये तवं खाइजे पाहे तवं निजीजे तैसे नव्हे कीं बाइ : पश्चात पाहारी उठीजे : तो सारस्वत काळु : देवता हृदयासि ये ।११४।

तुम्हां बोडी गंगा समानचि होआवी कीं गा ।११५।

मसनीचें वस्त्र घेपे : पोरें हो : तेयाचीचि वाट पाहाल : जें कव्हणी एकु मरों कां मरों कां : तरि कव्हणा एकातें हात च्यारि सुडा मागीजे ।११६।

तरि हात दोनि वेढीजे ।११७।

जेंचि अंथरुण तेंचि पांगरुण ।११८।

गांवाचां कडिसरा खांड देउळी एकी पाहीजे : तेथ श्वानकुंडली घालीजे ।११९।

तरि वोडबुरुडन्याओ कीजे : तें च्यारीचि घडीया कीं गा : तवं आपण निद्रा कीजे : पाठीं निजैलेयां मेलेयां सरीचि : पाहाटांची उठौनी फाळुकें झाडोनि जाओं लागिजे ।१२०।

तुम्हीं कां गा भिया : इयें तुम्हां आपणपें न दाखविती कीं : जरि हान भियाल तरि मां कें रिगीजैल ।१२१।

यें ऐसी उघडीं नागिवीं दिसते असति : परि यांची वास कव्हणाही पाहों न ये : यांही एकु अभिमानिये आति : ते जरि क्षोभलें होइल तरि मां काइ करीति ।१२२।

ऐसें आहाचपण न कीजे कीं गा ।१२३।

कां गा ऐसे म्हणाल : ऐसे न म्हणिजे कीं ।१२४।

पोरें हो : तुमचिया कडौनि तुम्हासि काइ न होआवे : एथौनि तुम्हां सकळही होआवें कीं ।१२५।

साधे हो : कव्हणातें काइ असे : हें सकळही देवाचें कीं : मा देवो दे तें आपुलें ।१२६।

ऐसें कैं होइल पोरां जीवां जें एथिचें तें आपुलें ।१२७।

भ्रमत भ्रमतां भेटि कां आपजौनि भेटि ।१२८।

एकमेकां भेटलेयां हें भेटलेयांचा पाडु जाय ।१२९।

श्रमनिवृत्ति कीजे ।१३०।

एकत्र खानपान कीजे ।१३१।

फाळुकीं पालटीजेति ।१३२।

एथीचिया कथा वार्ता कीजेति ।१३३।

सात पांच दीस एकत्र असीेजे : मग आपणालेयां जाओ लागीजे ।१३४।

पुर्वोत्तर क्षण दुर्लभ ।१३५।

तुम्हीं अच्युत गोत्रीय कीं गा : तुम्हां परस्परें परमप्रीति होआवी कीं ।१३६।

वानरेया : आतां म्हातारी तुम्हां लागे गा : कव्हणें अर्थें वंचिजे ना : बाइ : तुम्हीं यांते पुसिजे : पुरीकर नागदेवो ऐसें न म्हणिजे : हा एथीचिया जवळीका आन जाला असे ।१३७।

बाइ : परम प्रीति कीजे : परी विशिष्ठ बुद्धि न कीजे ।१३८।

प्रीति विखो होइजे ।१३९।

परस्परें मथन कीजे मग नूतन होए ।१४०।

स्वोक्ति : शास्त्रोक्ति : लोकोक्ति : इया तीन्हीं उक्ति परित्यजौनि एथीचिया व्युत्पत्ति व्युत्पन्नां होआवें ।१४१।

हें अध्रीत शास्त्र कीं गां ।१४२।

हें होउनि कवणाचें घर न रीगे : आपणचि याचें घर रिगीजे ।१४३।

यासी आपणपें दीजें तरि हें आपणपें दे ।१४४।

घेइजे : घोकीजे : मग काळें करौनि उपयोगा जाये ।१४५।

आतांचे तवं ऐसें घेपे : पाठीं अनुभवें जाणाल ।१४६।

हें वचन : हा वचनार्थु : वचन जाओं नेदीजे ।१४७।

काकु: कक्षा : भावार्थु : प्रकरण : जाणावें ।१४८।

महारवाडांहुनि धर्मु काढावा ।१४९।

विषयव्यवस्था करों जाणें तेयासिचि आइकों ये ।१५०।

अन्यवार्तेपसि निद्रा चांग : वरि निद्रा कीजे : परि अन्यवार्ता न कीजे ।१५१।

सुख : श्रेय : श्रृंगारु : इये तीन्ही नाहीं ते काइ गा गोष्टी : ते अरण्य रुदन कीं ।१५२।

तो तेतुला मंगळ काळु जाला ।१५३।

बाइ : तैसां स्थानीं तैसिया तैसिया गोष्टी तिया न कीजेति कीं ।१५४।

हां गा : हें तुमचें रहस्य कीं : मां आपुले रहस्य तें आणिका प्रति प्रकटीजे ना कीं ।१५५।

ऐसें न म्हणिजे कीं नाएका : लोकु डांकुली वाये तरिं तुम्हां अविधि होए ।१५६।

अन्यकथनें जाए ।१५७।

जेथौनि होए तेथीचिया विपरीत बुद्धी जाए ।१५८।

द्रव्य : यंत्र : सेवेतव जाए ।१५९।

अपक्व कखाइया जाए : तैसा कुयोगिया न वचे ।१६०।

विकल्पसंचारमात्रें जाए : तैसा विकारें न वचे ।१६१।

अपर धर्माचें धारण यातें विकल्पु बोलिजे ।१६२।

अनिष्ट सेेवेतव विकल्पु संचरे ।१६३।

अविधि अविधीतें प्रसवे : अविधीतव पुरुष धर्मापासौनि जाए ।१६४।

विद्यारुपें अविद्या संचरे : कां कृपारुपें संचरे ।१६५।

अविद्येची अनंतें द्वारें : अविद्या द्वार पांतुची असे ।१६६।

हें जाणिजे तेथ यथेष्ट चेष्टा न कीजेति कीं ।१६७।

प्रमाद चुकवीजे ।१६८।

विकल्पु स्फुरे तो अर्थु न कीजे ।१६९।

गुरुपसि अधिकवृत्ती न वर्तावें ।१७०।

हां गा : ऐसें ही करीतां लोकु न भजो : ऐसें न कीजे कीं ।१७१।

काम : क्रोध : लोभ : इये तीनि नरकद्वारें ।१७२।

वर्मस्पर्शे नरक : वर्मस्पर्शापसिं वरि शिरछेदु चांगु ।१७३।

पोरु जीवु आपुलें ओखटें गोमटें जाणें : परि जाणौनिचि करी ।१७४।

पोरु जीवु आपुलें मात्रागमन न देखे : परि पुढीलाचें वेश्यागमन देखे ।१७५।

बाइ : क्रियेसि अनुतापु पुरश्चरण ।१७६।

राग : द्वेष : काम : क्रोध : मद : मत्सर : यांचा त्यागु याचि नांवें संंन्यासु कीं बाइ ।१७७।

मनीचें अभिलाष निर्वतले तरि शुचि अशुचि इयें दोन्ही नाहीं नाएका ।१७८।

अहंतेचें मूळ सकंदी उपडावें नाएका : एर्‍हवीं अंकुर निगति: मग शांतीचे रोप लावावें ।१७९।

दोनि च्यारि भातवेळितीं जालीं तरी पोरा जीवाचे माथेया डोळे जाता वेळुचि न लगे ।१८०।

पोरा जीवा सामग्री जालीं तरि बाहीरि घालीतां वेळुचि न लगे ।१८१।

जीवु उपाधि क्षोभे ।१८२।

उपाधि मानीजे ।१८३।

निमित्त मानीजे ।१८४।

परमेश्वरु भज्यु : संबंधु वंद्यु ।१८५।

एथीचेयां संबंधा गेले तेतुले आटे गोटे आदिकरौनि तुम्हां नमस्करणीये कीं गा : मां येसनी देमती कैसेनी डावलीलि ।१८६।

आपुले सकळही निरोध परिहरौनि एथीचेयां संबंधीयांचेया संबंधा जाइजे : मग तो म्हणे तें कीजे ।१८७।

महात्मेनि प्रियवक्तेया होआवें ।१८८।

महात्मेनि सत्यार्जवा होआवे ।१८९।

बटिका एथ हांसतां ही लटिके न बोलिजे हो ।१९०।

तुमतें महारोग न बधिति ।१९१।

तुम्ही महात्मे कीं गा : तुम्हीं सकळही भोगें निमानावें कीं ।१९२।

कव्हणी एकु हिरडा बेहाडा दे : तरी ना न म्हणीजे ।१९३।

आपुलेया पडिलेयाची शुश्रुषा कीजे ।१९४।

पचन पाचन न कीजे ।१९५।

आपणेयापसी ज्ञाता विरक्ताचा संगु कीजे ।१९६।

धर्मा धरु दीजे ।१९७।

आर्ता प्रति बोलावें ।१९८।

लोहे पाणी गिळीलें ऐसे होउनि असावें ।१९९।

जो जें होउनि असे : तो तेचि बोले ।२००।

पोरु जीवु कांटेयाचेया पेटेयावरी विषयव्यवस्था करी ।२०१।

ईश्वरबोधें बोधवंतु : ईश्वरवियोगें देशाचा सेवटी झाडातळीं जन्म क्षेपों जाए : छर्दितान्न भोजनु करी हा ठावों वेर्‍ही घृणा जिंतिली : तो सवेतव एवं पर्यंत मैळे ।२०२।

उजळलेयाचेनि संगे मैळला उजळे ।२०३।

देखिलें गा : हा ठावों वेर्‍हीं देवतेचां ठाइं वेचेलपण होआवें ।२०४।

देवो माझा : मीं देवाचा : ऐसेया परि असीजो का देमती ।२०५।

मा तो एसणा परमेश्वरु : तेयाचेयां होइजो कीं देमती ।२०६।

दासोऽ हं न पुरे नाएका ।२०७।

तुम्हां गोविंदगती नाएका ।२०८।

हां गा रडिनलेयां काइ परमेश्वरु होईल : एणें खडकेसि कपाळ घेया पां जरि होए : तो जेया परि होआवा तेयाची परि होइल कीं ।२०९।

हें ही वरि होववेल पोरा जीवासि परि : एथौनि म्हणितलें तें न होववें ।२१०।

तैसें काइ देमती : जोन्हळे नाही तरि काइ अमेध्य पेवीं सों येइल : अन्न नाहीं तरि काइ विख खाओं येइल : तरि होत सांता हेंचि करीतां होइल कीं ।२११।

बाइ आर्ति करौनि एथीची दसीचि धरीली पुरे ।२१२।

साधे हो : जीव ऐसे अधोमुख असति : परमेश्वरु ऐसे वरि पडौनि पाहात असति : एकाधा एकु ऐसी वरौती वास पाहील तरि ऐसाचि काढीजैल ।२१३।

पांडेया : एथ अनुसराल तरि ऐसेंचि काढीजैल ।२१४।

ते एथीचि खावणी ।२१५।

हां गा एथ शरण आलेयां काइ मरण असे ।२१६।

बाइ : हें एथिचेया दर्शनाचिया चाडा येति असे तरि इयेतें नेणों कोणचि तारील ।२१७।

आर्तातें परमेश्वरु वाडोवाडीं गीवसीति ।२१८।

पोळलेया सुनेयांचेया परी असावें ।२१९।

दांत असति तेथ चणे नाहीं : चणे असति तेथ दांत नाहीं ।२२०।

ज्ञानेंविण वैराग्य तें काइ करावें बापेया ।२२१।

पोरा जीवा एथौनि म्हणितलें तें न होववें मग मरणा शरण जाए ।२२२।

वोखटें करीत सांता कव्हणी गोमटेयातें न पवे : गोमटें करीत सांता कव्हणी वोखटेयातें न पवे ।२२३।

पुरुष आपुलें करणीये करीत सांतां तेथचि कीर्तीते पावे ।२२४।

सत्यार्थ प्रकटे ।२२५।

बाइ : पोरेया एका कारणें सातां पांचांचा सासुरवासु साहीजे : मां विश्वाचा पति परमेश्वरु तेणेसिं धरणें मांडीलें असे तरि जगाचा सासुरवासु साहावा कीं ।२२६।

ऐसें न म्हणीजे कीं गा भोजेया : प्राणियासि असुख होए ।२२७।

ऐसें न म्हणीजे कीं गा डखलेया : पुरुषीं क्षोभु उपजैल तेव्हेळीं काइ कराल ।२२८।

ऐसें न म्हणीजे कीं गा डखलेया देवतेसि निरोधु होए ।२२९।

हां भट्टो : ऐसा सत्यही अर्थु न बोलिजे : मा एसणें असत्य तुम्हां कैसेनि बोलविले ।२३०।

ऐसें भय नुपजवीजे कीं ।२३१।

तुम्हीं महात्मे कीं गा : तुम्हीं चतुर्विधा भुतग्रामा अभय देयांवें कीं ।२३२।

वोखटें केलें : ऐसे कीजे : ऐसा प्राणिया धर्मापासौनि काढीजे ।२३३।

बाइ प्राणियांचेया इष्टा आड न एइजे कीं ।२३४।

बाइ प्राणियांचे दोष एथ न सांगिजेती कीं : एर्‍हवी एथौनि नेणिजे ऐसें काहीं असे : तरि कां म्हणीजे ना : करीति ना : परि एथिचेया दर्शनाही न एति ।२३५।

ऐसा अंतु न पाहिजे कीं ।२३६।

पुरुषाचेया ठाया जाइजे : तरि तेथिचें काहीं एक गोमटें घेउनि निगीजे ।२३७।

बाइ : महात्मेयां निरोधु न कीजे कीं ।२३८।

बाइ : हा प्राणियांचे चैतन्य पोळउनि आला असे : आतां एथौनि याचें मुख न पाहीजे ।२३९।

हिंसा वर्ते तिये स्थानीं महात्मेयां असों नये ।२४०।

हे करी तेया नरक : धरी तेया नरक : एक एकां दे तेया नरक ।२४१।

राजसे तामसे स्थानीं नसावें ।२४२।

एथीची सेवा ते भूतां अविरोधें करावीं कीं गा ।२४३।

विधि करीतां विधि स्फुरे ।२४४।

विधि अविधि परिहरे ।२४५।

एकु ग्रामपशु : एकु अरण्यपशु ।२४६।

एकु अमेध्यकृमी : एकु विषयकृमी ।२४७।

अमेध्यतुल्य विखो तो कवणासि दुर्लभु : जे कृमितुल्य पुरुष होती तेयासिचि कीं ।२४८।

एकु मळ त्यागिया : एकु अनिष्टत्यागिया : एकु स्वतेव जाए तेव्हेळीं सांडी ।२४९।

केतुले एक अनिष्ट उपसाहाति : केतुलेया एका अंतरायाते जनी ।२५०।

योगभ्रष्ट तो शाश्वतसम लोकांतें पावे ।२५१।

ऐसे गा जीव : जीवाचा वागारा नावेक ढिला करों न ए : जरि केला तरि आनौनी आन करितां वेळुचि न लगे ।२५२।

एथौनि वारिलें तें न कीजे कीं ।२५३।

तुम्हां एथौनि म्हणीतलें तें करणीये कीं गा ।२५४।

देमती देवें करावें तें केलें : पुरुषें करावें तें ठेलें ।२५५।

भोजेया : एथ अनुसरीजे ना तरि ऐसेयां अधोेपरीचेेया वृक्षा होइजैल गा : मग अनंत काळ दु:ख भोगिजतचि असीजैल ।२५६।

बाइ एथिचेया होइजे तै चुकेति ।२५७।

आप्ताचें वाक्य लागे ।२५८।

जीव झाडिले पुसिले वाहति त्रसले तरि जाति ।२५९।

उगे असा गा मंडळीका : जीव जीवां काहीं परिहारु असे ।२६०।

म्हातारी तमाचा मोटाळा गा : परि एकचि नीके जें एणें विण गतिचि नाहीं ।२६१।

सुखिया श्रीमंताचां घरीं पुरुषु होउनि जन्मति : मग बारा वरिखां संबंधु नेदुचि न ये ।२६२।

मार्गी प्रर्वतलेया पुरुषा अनुकूळा : प्रतिकूळा उदासा देवता होति ।२६३।

देश : काळ : वर्तमानाचें साह्य आति ।२६४।

रडिजे ना कीं गा : रडिनलेयां काजळी पडे ।२६५।

खरुज खाजवीना तो विरक्त ।२६६।

एथौनि म्हणीतलें तें म्हातारियेसिचि घडलें ।२६७।

अभोक्त्या म्हातारीचेया परी असावें ।२६८।

स्त्रियां सजन विजन होआवे ।२६९।

कवाडाचां देउळीं न बैसावें ।२७०।

समयस्थानी न बैसावें ।२७१।

साधा संताचिये वोसरीये बिढीर कीजे ।२७२।

बाइ : सुडे मागिजेति : मग आपुलें अंग झाकें ऐसें कीजे : आंचळे बहिर्वास न नेसिजेती ।२७३।

पृथ्वीएसणें रवण असें : आणि परमेश्वरा एसणी देवता असे मां चिंधी चिरकुटी नेणों कैसी मिळेल कीं ।२७४।

साधें हो : तुम्हीं आपली मांडी परपुरुषां देखो नेदिजे कीं ।२७५।

लज्जारुपधर्मु स्त्रियांसि रक्षण ।२७६।

महात्मे हो : एकली स्त्री तेव्हढेया वेळां तैसेया स्थाना न धाडिजे कीं ।२७७।

स्वयं दास्य तपस्विनाम् ।२७८।

सेवकासि अविधि नाहीं ।२७९।

हें देखिलें तैसे आठविजे : एथ नमस्कारु कीजे : सकृत सेवीजे : अति यत्न कीजे ।२८०।

हें असे तया रक्षण ।२८१।

अनुसरलेयाचिया दोनिचि घडिया पुरेति ।२८२।

एथ अधीन होइजे : एथौनि म्हणितलें तें कीजे तें अनुसरण ।२८३।

स्वस्थे चित्ते बुद्धय : संभवन्ति ।२८४।

एथचेयासि प्राणपदीचि पीडा नाहीं ।२८५।

तैसें काइ बाइ सेवटीली च्यारी घडीया ऐसीया गेलीया तरि काइ : ना तैसिया गेलीया तरि काइ ।२८६।

समान वायोसि लिंगदेहाचें उत्क्रमण यातें मृत्यु बोलिजे ।२८७।

हां गा : जन्माचीया काइ अनुपपत्ति पडिलिया असेति : ऐसीं अनंतें जन्में वायां गेलीं : म्हेळिये जन्म दीधलें : मोळियें जन्म दीधलें : सिवणेया जन्म दीधलें : रुंभणेया दीधलें : मां हें एक जन्म एथ दिजो कां : मां काइ होइल तें पाहिजो ।२८८।

बाइ : ब्रह्मानुभूती ते दुर्लभ।२८९।

आचार माळिका

स्वातंत्र्यु हा मोक्ष : पारतंत्र्यु हा बंध ।१।

मायाबहिनी आशंसिति आमचा दांडिये डोळिये कै जाइल : परमेश्वरु आशंसिति कटंकटीयेमान गव्हरीं कपाटी ऐसा कै असेल ।२।

पोरा जीवा कव्हणी सामग्री न होआवी ।३।

पोरा जीवा सामग्री म्हणिजे गडदरागा ।४।

सकळही सामग्रिया परित्यजौनि एथिचिया व्युत्पत्ती व्युत्पन्ना होआवें ।५।

एषणात्रय त्यजावें ।६।

लोकीचा श्रेष्ठ तो एथीचा नष्ट : एथीचा श्रेष्ठ तो लोकीचा नष्ट ।७।

जवं जवं प्रयोजकु तवं तवं अप्रयोजकु : जवं जवं अप्रयोजकु तवं तवं प्रयोजकु ।८।

तवं पर्यंत असावें जवं आपुलें श्रेष्ठत्व नष्टत्व प्रकाशेना ।९।

अन्यधर्मे पुरुष धर्मापासौनि जाए ।१०।

विकारें पुरुषु धर्मापासौनि जाये ।११।

आपणेयां उपद्रो उठी तेयाचा असंबंधु कीजे ।१२।

ऋष्यश्रृंग स्त्रीपुुरुष ऐसा विभागुही नेणें : परि संबंधे पोराचें चैतन्य वेधलें ।१३।

द्रव्यसंबंधा साक्षी न ठाकावें ।१४।

क्रय विक्रय न करावें ।१५।

छेद भेद न करावें ।१६।

तुम्हां आवडी नावडी न होआवी ।१७।

तुम्हां विधिनिषेधु न होआवा ।१८।

तुम्हीं जात्रे क्षेत्रा न वचावें ।१९।

तुम्हीं पुरीं पाटणीं न वसावें ।२०।

आपण कव्हणाची ताति न कीजे तरि आपुली कव्हणी न करी ।२१।

भांडते दोघै अन्यायी ।२२।

पोरा जीवा अप्राप्ति ते दुर्लभ ।२३।

पोर जीवु करीना : परि करीन ऐसी हांव ही न बंधे ।२४।

पोरा जीवा आग्रहो नाहीं : आग्रहो जरि होए तरि परमेश्वरु ज्ञानियासि प्रेम देती ।२५।

एकु सेवित जाए एकु न सेवित जाए ।२६।

हें शास्त्र कव्हणाप्रति न बोलावें ।२७।

अशक्तासि विधि नाहीं ।२८।

हां गा : एथौनि म्हणावें तेव्हेळीं तुम्हीं करावें कीं हें तुम्हासिची स्फुरावें ।२९।

तूं उभा ठाकलासिचि पुरे : सिद्धीते एथौनिची नेइजैल कीं ।३०।

श्रीप्रभुचियेे सेवे आणिकां द्रव्याचें : ना : तूं नसुधाचि पुरेसी ।३१।

हां गा एथीचा व्यापारु तो एथ पुसों पुसों कीजे कीं ।३२।

पांडेया : हें तुम्हां दारिचेया खुंटेया इतुलेही न पढिये गा ।३३।

मंडळीका : हें तुम्हां काळें करौनि वस्त्रतुल्य होइल गा ।३४।

अधिकरण भज्य होए : पूज्य होए ।३५।

एकु देवो एकु धर्मु तया परस्परें परमप्रीति होआवी कीं ।३६।

हां गा : आणिकां साभिमानियांचेया होआवें मा एका निराभिमानियाचेया होइजो का : हें तरि निराभिमान परी आपुलेया लागुनि साभिमानियांचे साभिमान ।३७।

हाती गुढरिला नीका दिसे : जोगी मुंडिला नीका दिसे ।३८।

सवे ते काइसेयाची ही न होआवी ।३९।

अटन तें निरुद्देशें करावें ।४०।

गांउनि गांवा जावें ।४१।

सवेळेचा पाहारीं भीलाचिये पाळीये भय नाहीं ।४२।

झाडातळीं विजन कीजे ।४३।

अति असुरे स्थानीं नसावें ।४४।

असावें ऐसें तोंडावरि फाळुकें घालौनि ।४५।

नामस्मरण करावे : नामीं अभिमानु आति ।४६।

वस्तुस्मरण तेंही एक स्मरणचि कीं गा ।४७।

निरंतर परमेश्वराची आर्ति करावी ।४८।

निरंतर परमेश्वराचा विरहो भावावा ।४९।

किंचित दु:खे पाप नासे : आनंदें पुण्यराशी नासेति ।५०।

परमेश्वरा लागीं रंका होआवें ।५१।

निर्धूम नगर जालेया भिक्षे रिगीजे ।५२।

तिसरा पाहारीं भिक्षा कीजे ।५३।

पाठीं भिक्षे जाए तो विरक्तु ।५४।

एकी भिक्षा न करावी ।५५।

क्षण एकु उभेयां राहिजे मग निगौनि जाइजे ।५६।

भिक्षे पडिला हातीं घेइजे ।५७।

आला विखो पायें हानावा ।५८।

भिक्षा मागौनि नदीचां तीरीं भोजन कीजे ।५९।

क्षुधानिवृत्ति कीजे ।६०।

जेवितां आवडी गोडी न होआवी ।६१।

तुम्हां घृणात्यागु होआवा ।६२।

हां गा : हें भिक्षान्न कीं : हें पवित्राचे पवित्र कीं ।६३।

तुम्हीं महात्मे कीं गा : तुम्हीं पात्रीं पडिलें भोजन कीजे कीं ।६४।

मंडळीका : धाये तवं अन्न घेइजे गा : अन्नीं काइ दोषु असे : त्रसाल तरि खळया पाळी टेकाल ।६५।

गांवा बाहिरीं खांडदेउळीं कां झाडातळीं निद्रा कीजे ।६६।

स्वभावीं वर्तीजे ।६७।

शीतोष्णपरिहारु कीजे ।६८।

वोली बहिर्वास न नेसावें ।६९।

मार्गी पडिलें वस्त्र घेपे ।७०।

श्वानकुंडली का कोपरदुटी पांंगुरावी ।७१।

तुम्हां अशन आच्छादन होआवें ।७२।

नवें वस्त्र न नेसावें ।७३।

मळघाणी फेडिजे जळघाणी लाविजे ।७४।

गोंडीबुंथी घालीजे ।७५।

अन्यमार्गियांचा संसर्गु न करावा ।७६।

अन्यमार्गियांचिये गुंफे न वचावें ७७।

गोरुवा सरिसे गांवांतु रिगीजे ।७८।

देउळीं निद्रा न कीजे ।७९।

एकां वेडीं पिसीं नुरेति ।८०।

सवेळेचि उठौनि मार्गी उभया राहिजे : सांगातु पाहिजे : मागिले म्हणेति पुढिलांची: पुढीले म्हणंति मागिलांची ।८१।

सिंवनी चिंचना आपली मोळी गांवा बाहिरीं झाडातळीं घालावी ।८२।

गंगातीरीं असीजे : सातां पांचां दीसा आपली दावली दाविजे ।८३।

अपरधर्मु सांडिला : परधर्मु तरि आतिचिना तो योगभ्रष्ट बोलिजे ।८४।

वियोगीं देह पाडी तेयाही एक भलें आति ।८५।

परधर्म तुमचें जीवन कीं गा ।८६।

पुन:संबंधिया पुरुषु सुखियाश्रीमंताचां घरीं जन्मे : विषयव्यवहारावरि बुद्धि नाहीं : देवतां तीर्थांवरि बुद्धि नाहीं : हें नव्हें : हें नव्हें ।८७।

हृदयशून्यां धर्मी अधिकारु नाहीं ।८८।

भक्तिजडां धर्मी अधिकारु नाहीं ।८९।

कर्मजडा धर्मी अधिकारु नाहीं ।९०।

कर्मचांडाळापसि जातिचांडाळु तो चांग ।९१।

देमती : नाथु असे : परि जीवें अनाथां होआवें कीं ।९२।

देमती : कृपाळु असेे : परि जीवें कृपेसि पात्र होआवें कीं ।९३।

देमती : दानी असे: परि जीवास आर्ती होआवी की ।९४।

देमती : वज्रपंजरु असे : परि जीवें शरणांगता होआवें कीं ।९५।

विखो न सेवी तरि जीवु काइ गा मरतु असे : निरोधु साहिलेया अधिक योग्यता होए : दक्षता होए ।९६।

विखो अनंता तापासि आश्रयो नरकासि हेतु ।९७।

राजस : तामस : सात्विक त्यागु ।९८।

देवो तो देवेंचि पाविजे ।९९।

हें स्मरे तो दुर्लभु ।१००।

बहुतांमध्यें जयाची विशेष प्रतीति तेयापासौनि सुख स्वास्थ्य जालें ऐसें जाणावें ।१०१।

एकु विधिकिंकरु : एकु परमेश्वरुकिंकरु : आपण विधिकिंकरा न होआवें : परमेश्वर किंकरा होआवें ।१०२।

विधिभावातीता होआवें ।१०३।

संग्रहो करावा।१०४।

हिंसा न करावी ।१०५।

हिंसा पाप : पापातव नरक ।१०६।

परद्रव्याचेनी नांवें पडिला कवडा आदिकरौनि नेघावा ।१०७।

प्राप्तविषयांचा त्यागु करावा ।१०८।

मळवतयागु तो निर्गुण ।१०९।

परमेश्वरालागीं काळ : कुळ : लोकु : ये तीन्हीं आंगवावी ।११०।

तळहातें भाली हानावी ।१११।

समर्थो धर्ममाचरेत् ।११२।

विरळा नांदे ।११३।

सेंवतेयांचे आरळ खुंपे तोचि खांडेया भालेयाचे घाये आढें देवढें साहे ।११४।

जीवापासौनि पुरुषु उठी तेयाचें नेणों कोणचि सिद्धी ने ।११५।

पुरुषु आपुलें करणीये करौनि असावें : मग दैवें करावें ते दैवचि करील ।११६।

पुरुषु जेतुल जेतुला प्रयत्न करी तेतुल तेतुलें दैव साह्यातें करी ।११७।

एकु शास्त्र जाणौनि बोले : एकु नेणौनि बोले ।११८।

परमेश्वरशास्त्रीं एवं वा नाहीं : पर्याय नाहीं : अभावी भावरुप फळ नाहीं : अकरणी प्रत्यवायो नाहीं ।११९।

परमेश्वरपरायणाचेनि संबंधे सुख : श्रेय : ज्ञान : इये तीन्ही होति ।१२०।

अक्षरदाता जीव : बोधदाता परमेश्वरु ।१२१।

जिज्ञासा करावी ।१२२।

आचारें योग्यता : विचारें द्दढता ।१२३।

भाजिये गुरु मानावा ।१२४।

गुरु विश्वासु म्हैसेयाचा ।१२५।

गुरुरनुगंतव्य ।१२६।

निरालंबी एकलेयां काळु क्षेपवेना ।१२७।

देवतेसि निरोधु होए ऐसी सेवा न कीजे ।१२८।

ईश्वरीं जीव निल्लळ होती ।१२९।

पुरुष सवेतव जाए ।१३०।

एकला विजया ।१३१।

नित्याटन करावें ।१३२।

ग्रामे एकरात्रं नगरे पंचरात्रम् ।१३३।

पुरुषोत्तमक्षेत्रा न वचावें ।१३४।

सात्वीक स्थान सेवावें ।१३५।

प्रसन्नता ऐसा एकु धर्मु ईश्वरीचि आति : संबंध व्यक्त : व्यक्ते संबंधीं अव्यक्त प्रसन्नता परमेश्वरु निक्षेपीति मग आपुला संबंधु देति म्हणौनि संबंधु वंद्य ।१३६।

देवो गीति गावों नेणिजे गा पांडेया ।१३७।

देवाची आर्ति होआवी : मार्गाची प्रीति होआवी : विषयांची निवृत्ति होआवी : देहाची अनास्था होआवी ।१३८।

आपुली मळीन माती करावी ।१३९।

कोमळज्ञानीं आचारें इंद्रियधर्म पातळ होति : अपरोक्ष दशे जाति ।१४०।

तिळ खाइजे तिळा इतुली भूक जाए ।१४१।

चना विखो ।१४२।

शौचाचारं न शोधयेत् ।१४३।

सारविला उंबरा दखोैनि साउमेंया न वचावें ।१४४।

पूजा पुरस्कारु न स्वीकरावें ।१४५।

स्तुतिपर वैराग्य न करावें ।१४६।

सामान्यस्त्रीसदनस्थिता पासौनि संदेहत्रुटी करुं ए ।१४७।

जरठावस्थे विकारु जाए : परि विकल्पु न वचें ।१४८।

वेडेयां : पिसेयां : बधिरेयां : अंधा : मुकेयां धर्मी अधिकारु नाहीं ।१४९।

नित्यानित्यवस्तुविवेक : शमदमादिसाधनं मुमुक्षत्वं च ।१५०।

जोचि अवसरु विषया तोचि ब्रह्मविद्ये ।१५१।

आर्तो दु:खत्रसच्चित्तो जिज्ञासुरुतमो मयि : अर्थार्थीं नित्यपदाकांक्षी श्रद्धया परमास्तिक ।१५२।

दर्शनीं अदर्शन : अदर्शनीं दर्शन ।१५३।

संबंधी असंबंधु : असंबंधी संबंधु ।१५४।

परमेश्वरु अमूर्त : ब्रह्मविद्या अमूर्त : परमेश्वराप्राप्तिफलरुप अमूर्त ।१५५।

समानधर्मीं पतिप्रन्न ।१५६।

आपणेयांतें काहीं असे तरि तैसें कीजे : ना तरि आपणेयां सांगातें भिक्षे नेइजे ।१५७।

आग्रहो परिग्रहो न होआवा ।१५८।

कोपिनोद्धारण करावें ।१५९।

धौ विकारु ।१६०।

संपन्नाचां घरीं न वसावें ।१६१।

विषयाचा संगु न करावा ।१६२।

पोरा जीवा जेंचि पोटी तेंचि तोंडी ।१६३।

पोर जीवु आपणेयां करों आवडे तेंचि करी : पाठी पुढीलावरि ठेवी ।१६४।

आळोंचु न करावा ।१६५।

अविद्यारुपें अविद्या संचरे ।१६६।

कलियुगीं पतिव्रता हा धर्मु न वर्ते : अवकाशु नाहीं : प्रार्थिता नरु नाहीं : म्हणौनि चुके ।१६७।

जेतुल जेतुला प्रपंची जडे तेतुल तेतुला ईश्वरीं उजडे : जेतुल जेतुला ईश्वरीं जडे तेतुल तेतुला प्रपंची उजडे ।१६८।

सर्वत्र विषयीं गोडी तो रसु ।१६९।

गीत विखो ।१७०।

बाहिरीं बैसलेयां परतौनि पाहिलेयां अमेध्यो विकारु ।१७१।

अपेक्षापूर्वक अन्यधर्माचा अंगिकारु यातें विकल्पु बोलिजे ।१७२।

परमेश्वर आज्ञा अतिक्रमी तरि माया बोधु हरी ।१७३।

देहपर्यंत पुरुषु उरलाचि असे ।१७४।

असे तवं करी पडेतवं धावें ।१७५।

कराल तरि होइल कीं गा ।१७६।

एथीची मोहरचि पुरे ।१७७।

जीवु सर्वाधिकार्या ।१७८।

स्वत: वैराग्य तें दुर्लभु ।१७९।

ब्रह्मचर्य त्रसले जीव विषयाकारणें मढीं उपरतीति ।१८०।

तेणें तंव मोकलावें मग जयाचें ते सांभाळीति ।१८१।

खैरीं काढीला महात्मा विकरेना ऐसेया होआवें ।१८२।

ऐसिया एसिया गोष्टी तिया रानीं वनीं कीजेति कीं ।१८३।

तेल : तूप : हा गाळोत्तर रसु कीं गा : मा रसा काइ साक्षी ठाकीजताए ।१८४।

मूर्तिस्मरण तें वस्तुस्मरण होए : नामस्मरण तें वस्तुस्मरण होए आणि नामस्मरणही होए ।१८५।

आपुले निरोध आणिकां द्योतविजेति तेंही एक करवणेंचि कीं गा ।१८६।

विजना जातां जैसा काहीं खळिये बैसावेया नेइजताए ।१८७।

जीव : देवता : परमेश्वर आणि प्रपंचु हें जैसें असे तैसें जाणिजे : इतुलेनि कपाळावरि फाळुकें घालौनि जावों लागिजे जेथ गेलेया मेलेयाची सुधि न लाभे ।१८८।

पडे तवं धावावें मरे तवं करावें ।१८९।

एकी कोंठाचे बैल भांडति जुंझेति : परि कुसीसि कुसी लाउनि बैंसेति ।१९०।

ओठांतरां निगालें तें देशांतरां जाए ।१९१।

जीव स्फटिकाकारु ।१९२।

जयातेंचि वंदिजे तेयातेंचि वाणिजे ।१९३।

दृष्टें अदृष्ट न साधे ।१९४।

अनुसरलेयातें निरोध न बधीति ।१९५।

परमेश्वराची प्रवृत्ति भंगो नेदावी: परमेश्वरपरायणाची मनोवृत्ति भंगो नेदावी ।१९६।

दोही ओठां टाळें घालावें ।१९७।

जाणौनि नेणें तो सदैवु ।१९८।

झाडितां झडेना : तोडितां तुटेना : तेया प्रति बोलावें ।१९९।

व्यायामें वायु तुटे ।२००।

अनुसरलेयाचा देवो ।२०१।

निर्हेतुक भिक्षा ।२०२।

अवेळ : सवेळ : दुपाहार इये तीन्हीं चुकवावी ।२०३।

सिंदी : नदी : दंदी : यांचेया गावीं न वसावें ।२०४।

फळतेया फुलतेया वृक्षाखाली न बैसावें ।२०५।

उडत पांखीरु न पाहावें ।२०६।

पाखरिला घोडा न पाहावा ।२०७।

रसज्ञान : क्रियाज्ञान : वृत्तिज्ञान : इये तीन्हीं वाढविलीं तेसनी वाढतीं ।२०८।

इन्द्रियार्थरसव्यतिरिक्त अन्नीं काइ गा दोषु असे ।२०९।

आपुलें सीत मुंगिये आदिकरौनि नेवों नेदावें ।२१०।

भिक्षान्नीं सवावो पळ अमृत आति ।२११।

पोर जीव पडनी घालों धांवें : आगीं रिगों जाये : आणि सीसनिये भिये ।२१२।

पुरुषु प्रमाद चुकवी तरि चुके ।२१३।

बाइ : ऐसा कवण मायेचां पोटीं उपजैल जो देशावरि खांडे काढौनि जाए ।२१४।

बाइ : ऐसा एकला पळपळीत निग जो दु:खपुर्वक आपुलेया देवातें आठवीत वर्ते ।२१५।

धड तुटे तवं धावावें : देह पडे तवं करावें : परि जीवन न सोडावें ।२१६।

चित्त : वित्त : काया : इए तीन्हीं परमेश्वरासि अर्पावी ।२१७।

देशाचा सेवटां : झाडाचां तळवटां : देवळाचां कोनटां : ए तीन्हीं सेउनि असावें ।२१८।

आपण उपयोगा न वचे : आणिकातें पुढासूनि विखो भोगी ।२१९।

विकारु विखो न सेवावा ।२२०।

धीरवीरां होआवें ।२२१।

अखोरी अबोटी जन्म परमेश्वरासि देयावें ।२२२।

एथीचा परमाणु तो ब्रह्मादिकांसि दुर्लभु कीं ।२२३।

विधि करीतां अविधि संचरे ।२२४।

तीसरीये धारे वरि चाले ऐसा कवण असे ।२२५।

वोथंबा डावो देखें ।२२६।

पळे तो परतो लाहें ।२२७।

नागवेया नसावें ।२२८।

हे सृष्टी अति दीर्घ गा ।२२९।

एथौनि निरय निरुपीजेति तरि देह धरी ऐसा कवण असे ।२३०।

हातीं शेळी डोइये पाळणा : ए काह्याची रे : म्हणे नागदेव राउळाची : एथीचें नांव ही नाहीं ।२३१।

आपुलेयातें परमेश्वरु हाळो हाळियां गीवसीति ।२३२।

पुरुषा होउनि असावें ।२३३।

केवळ घरैतें वरैतें तेथ नसावें ।२३४।

कारुवा तारुवांचेया सारिखें वस्त्र होआवें ।२३५।

बुडी मात्र दीजे ।२३६।

दीमीळणीं गांवाचां कडिसरां टेकावें ।२३७।

बाइ : हें बरवें नव्हे : भितरीं असे तें बरवें ।२३८।

सकळही संपत्ति तिया असंपत्ति : जीवाची परमेश्वरसंपत्ति ।२३९।

देवासि पढीये तेयापसि देवपढिये तें नीके ।२४०।

राजसां तामसां धर्मी अधिकारु नाहीं ।२४१।

सन्निधानीं असौनि अधीनता नाहीं यातें असन्निधान बोलिजे : असन्निधानी असौनि आज्ञाप्रतिपाळण करी यातें सन्निधान बोलिजे ।२४२।

बाइ : तें यातना देह कीं ।२४३।

हें शास्त्र दृष्टाप्रति कारण होए : कां अदृष्टाप्रति कारण होए ।२४४।

देखिली श्रीमूर्ति आठवीजे ते मुर्तिस्मरण बोलिजे : आइकिलीेश्रीमूर्ति आठविजे ते भावना बोलिजे ।२४५।

प्रतिक्षणीं प्रतिक्षणीं त्यागु ।२४६।

भलतैसा : भलतेथ : भलतेया परी परमेश्वरु एकु आति ।२४७।

ज्ञान : ध्यान : भजन हे परमेश्वरींची आति ।२४८।

विकल्पत्रुटि अन्य महात्मेयासि साजात्य ।२४९।

संयोग : वियोग : साह्य : इए तीन्हीं माया करी ।२५०।

स्त्री म्हणिजे मेंढियेचें खाजें गा ।२५१।

मीठाचा खडा आदिकरौनि विखो ।२५२।

जोचि रोग : तेचि औषधि : तेचिं देवता : इए तीन्हीं मिळेति तरि जिए ।२५३।

दुना पथ्य घेइजे ।२५४।

विचार

चैतन्य : माया : विदेह : परा : शक्ति : ।१।

मुख्य : तुल्य : अंश : रश्मि: ।२।

दिव्य : दिनस्थ : जळस्थ : स्थळस्थ : ।३।

घटघटाभिमानीए अनुस्यूत ।४।

जड : अजड : चिज्जड : एवं त्रिविध माया ।५।

मायाचक्रभ्रमणन्यायें पर्यायवशें जीव सकळही पावे एक वांचौनि ।६।

अच्युताचां अज्ञानीं जन्मलें जें अन्यथाज्ञान तें स्वर्ग : नरक : कर्मभूमि : मोक्ष : या चतुर्विधातें आपजवीति असे ।७।

चैतन्यानुभूतीं जीवीं बीजभूत अविद्या असे : जैसा वटबीजीं वटु असे ।८।

चैतन्यप्राप्तीं जीव भ्रांत होउनि असे ।९।

चैतन्यविद्ये चैतन्याचे मळ जाति : जीवाचे आच्छादेति ।१०।

जीवाचां मोक्षीं चैतन्या मोक्षु ।११।

चैतन्यफळ मोक्षतुल्य : परि तेथ च्युति हा दोषु ।१२।

चैतन्यफळादिकरौनि सकळही फळें साधनें मूर्ते ।१३।

कव्हणीं एकु वेद विभागु चैतन्याचेया अस्तित्वातें जाणें तथा बोले ।१४।

परा : पश्यंती : मध्यमा : वैखरी : ।१५।

चैतन्योद्भवा परा : यंत्रोद्भवा वैखरी ।१६।

वैखरीद्वारें परेचा व्यापारु ।१७।

सिद्धांचां ठाइं पश्यंती : देवांचां ठाइं मध्यमा : मनुष्याचा ठाइं वैखरी : ।१८।

नाद : बिंदु : कळा : ज्योति : लो : अवस्था : आनंदु : आवेशु ।१९।

कराळी : विकराळी : उमापति : पशुपति : सदाशिव : ब्रह्मा : विष्णु : महादेव ।२०।

आगम ते सिद्धकृत ।२१।

स्वच्छंद : लळित : मंथान : भैरव : आदि : कादि ।२२।

सृष्टिकर्ते ते : अहंकारीए ते : अभिमानीए ते ।२३।

आणवीं साध्य : साधन : साधकु ऐसी त्रिविध प्रतीति ।२४।

शाक्तेयीं साध्य : साधकु : ऐसी द्वीध प्रतीति ।२५।

शांभवीं साध्य प्रतीति ।२६।

आवेशु : अवस्था : स्थिति ।२७।

साजात्यां साजात्यां वेधु ।२८।

अंशवेधे संपुर्णाची प्राप्ति ।२९।

वेधु : विद्या : ज्ञान : साधन : धर्मु : ।३०।

उभयान्वीक्षमाणे सत्ता संन्निधिमात्रें शक्ति संचरे यातें अंतर्यागु बोलिजे ।३१।

जो जेतुला तो तेतुलेयाते करों सके ।३२।

चंद्रसूर्यापासौनि आणव ।३३।

इंद्रादिकां अनात्मा आत्मप्रतीति : ब्रह्मादिकां अत्मा आत्मप्रतीति ।३४।

तीन्हीं ग्रामें : तीन्हीं राणें समानचि ।३५।

ब्रह्मविद्येची सारखी वाटणी इतर विद्यांचा ठाइं आति ।३६।

मोक्षमार्गी जीवाचें मळ जाति : विद्यामार्गी आच्छादति ।३७।

देहविद्ये देह प्रकाशे : देहीं भूतें तन्मात्रापर्यंत उरेति ।३८।

सातै अंश झडेति : एकाकार वृत्ति उरे ।३९।

हेमकाया होए : वज्रकाया होए : ।४०।

गत्वा मुक्ति ।४१।

लौकिक : भौतिक : उपसंहाराख्य : लाभाख्य : ।४२।

उच असे तो तळीचील देखे ।४३।

पैल एैलाते व्यापौनि वर्ते : ऐल पैलाचें अस्तित्व जाणें ।४४।

नित्यबद्धा देवता : बद्धमुक्त जीवु: नित्यमुक्त परमेश्वरु ।४५।

जीवें जें होआवें तेंचि होए :देवता हों जाणें : उरों जाणें ।४६।

देवतेंसि लीलाविग्रहधारित्व आति ।४७।

देवता आड रिगौनि फळातें दे ।४८।

प्रल्हादाचेया भावातिशयें आणिके देवतें आड रिगौनि फळातें दिधलें ।४९।

भावों अनियत फळद ।५०।

प्रल्हादाचेया कैवारा भूत एक अवतरले : तेयाचेंया कोपापरौतें चराचर विश्व न पुरे ।५१।

भावासि काइ रावो राणा असे ।५२।

भावो व्यभिचारिया ।५३।

एकु विधि अनुष्ठान : एकु भावो अनुष्ठान : हें विधिभावातीत अनुष्ठान ।५४।

एकु विधिसहितभावो : एकु भावासहीत विधि ।५५।

तुर्य ब्रह्म तें ।५६।

विश्वाख्यप्रपंचीं भूतें तन्मात्रा ।५७।

धातु : भूतें : तत्वे : याचियां तन्मात्रा तो सूक्ष्म प्रपंच बोलिजे ।५८।

सकळही प्रपंचु पंचभौतिकु त्रिगुणात्मकु ।५९।

एकांतें प्रपंचु परिणमु बोलिजे : एकांतें विवर्त बोलिजे : एथ तैंसे नव्हे एथ प्रपंचु म्हणिजे स्वतंत्र पदार्थु एकु आति : जैसा गंधर्वनगरीचा हस्ती ।६०।

अविद्या अविद्येसि कारण : प्रपंचु प्रपंचासि कारण ।६१।

यदनन्तरं यद् दृश्यते तत्तस्य कारणम् ।६२।

प्रपंचु व्यभिचारिया ।६३।

जेंचि मान ब्रह्मांडी तेंचि पिंडी ।६४।

जियाचि देवता ब्रह्मांडी तियाचि पिंडीं ।६५।

ब्रह्मांडी मूर्ता देवता : पिंडी मेदमांसरुपा देवता ।६६।

हरीता : पीता : श्वेता : मंजिष्ठा : कृष्णा: मूर्ति : प्रकाशेति ।६७।

पिंडहनन केलेयां ब्रह्माण्डहनन केलें होए ।६८।

आत्मघातकियासि कव्हणीं ठावो नाहीं : नाएका ।६९।

मन : बुद्धि : चित्त : अहंकारु ।७०।

मन अधिष्ठात्री देवता ब्रह्मा : बुद्धि अधिष्ठात्री देवता शेषशय्या : चित्त अधिष्ठात्री देवता विष्णु : अहंकार अधिष्ठात्री देवता महादेवो : चक्षु अधिष्ठात्री देवता सूर्य ।७१।

युगाविपरीत धर्मानुष्ठानवंता कृतयुगी जन्म ।७२।

ज्ञान : ध्यान : मंत्र : स्थावरु ।७३।

तेथिचे ते सकळही आत्माराम ।७४।

उपदेशिला बहिर्यागु हीने युगीं संभवे ।७५।

भक्ति जैसी रामाची लक्ष्मणे केली ।७६।

राम शुद्ध सत्त्वाचा अवतारु ।७७।

श्रीकृष्णीं अष्टविध ब्रह्मचर्य ।७८।

बाइ : विखो तो श्रीकृष्णचक्रवर्तीचि भोगों जाणेति ।७९।

श्रीप्रभु आर्य : हें आर्यनागर : श्रीकृष्णचक्रवर्ती ते नागर ।८०।

पूर्वी श्रीकृष्णचक्रवर्ती विज्ञानशक्ति खेळीनले : पाठीं ज्ञानशक्ति स्वीकरिली ।८१।

बाइ : श्रीकृष्णचक्रवर्ती अनंत उद्धरिले : परि उद्धवार्जुन हे पुराण प्रसिद्ध ।८२।

बळिभद्रीं साजात्यां वेधु : श्रीकृष्णीं साजात्या वैजात्या वेधु ।८३।

एकु आज्ञाधरु : एकु पराज्ञाधरु ।८४।

संजयो चतुर्भूज देखे : उद्धवदेवो कैवल्य परब्रह्म देखति : अर्जुन विश्वरुप देखे ।८५।

अर्जुन सकळ सामग्रिया सामग्रीवंतु : एक वांचौनि ।८६।

पूसतेया अर्जुनाचिया युक्ती श्रीकृष्णचक्रवर्तीसिचि परिहरेति ।८७।

पूसतेया अर्जुनाचा अभिप्रावो ऐसा जे श्रीकृष्णचक्रवर्तीचे परावरत्व सुचौनि बोलतु असे ।८८।

बाइ : श्रीकृष्णचक्रवर्ती तिये तिये शक्तीचां ठाइं अहंता स्वीकरुनि ते ते शक्ति आत्मत्वेंचि निरुपितें जाले ।८९।

श्रीकृष्णचक्रवर्ती निरुपिता अर्जुनाची शंका पारुखेचि ना तेव्हेळीं अपरोक्ष प्रकटिलें : मग शंका पारुखली ।९०।

श्रीकृष्णीं शतां सहस्त्रां वेधु : एथचां नामींचि वेधु ।९१।

जेचि गति यशोदे तेचि पुतनेसि ।९२।

कर्पुरकज्जलिकावत् श्रीकृष्णचक्रवर्ती पुरत्यागु केला ।९३।

कलियुगीं विद्यावंत नाहीं ।९४।

कलियुगीं आगम न वर्तेति ।९५।

उत्तरापंथीं तंत्रशेषु वर्ते ।९६।

कलियुगीं विखो आति परि विषया नाहीं : हे राये मंडळीक आदिकरौनि सकळही निरोधु परिहारिये कीं गा ।९७।

कलियुगीं तस्करत्व पुंश्चळत्व हा धर्म स्वरुपीं न वर्ते ऐसा विरळा ।९८।

कलियुगीं जन्ममात्रे चौकडी पर्यंत नरक ।९९।

युगाविपरीत धर्मानुष्ठान घडे तया चौकडी चुके ।१००।

त्रेतीं सीता बरवी : द्वापरीं श्रीकृष्णचक्रवर्ती बरवें : कलियुगीं हे बरवें ।१०१।

एथिचेया सारिखे अन्यत्र आति : परि एथिचि दैवी ते क्वापि नाहीं ।१०२।

एथिचेया सारिखेया पाठीं धावाल ।१०३।

परमेश्वरु त्रिविधापरि मायापुरातें स्वीकरीति : ।१०४।

गर्भी स्वीकरीति : कां पतितातें उठवीति : कां जीवातें दवडीति ।१०५।

देवो कच्छ होति : मच्छ होति : देवामध्यें अवतरति : नरामध्यें अवतरति : तिर्यंचामध्यें अवतरति : नरामध्यें अवतरला सांता देवो वेडा होए : पीसा होए : मूका होए : परि चालता बोलता देवो तो दुर्लभु कीं गा ।१०६।

परमेश्वरु सकळही देहधर्म जीवधर्म स्वीकरीति ।१०७।

एथिचें सकळही तें एथिचियाचि प्रवृत्ती ।१०८।

युगानुरुप परमेश्वरु क्रीडति : परि किंचिद्विलक्षण ।१०९।

परमेश्वरु स्वयं ज्ञानशक्ति स्वीकरीति : कां मायापुरातें आदिकरौनि स्वीकरीति ।११०।

वेधौनि बोधीति : कां बोधौनि वेधीति ।१११।

जीवासि बोधु आणि बोधशक्ति तें एथौनिचि कीं गा ।११२।

अधिकरणातें अधिकरौनि परमेश्वरुचि उद्धरीति ।११३।

एथ प्रवृत्ति जाली तरि स्तंभा एकातें निमित्त करौनि उद्धरीजैल ।११४।

एथ प्रवृत्ति जाली तरि स्तंभा एकातें निमित्त करौनि बोलिजैल ।११५।

आद्यंतीं दैव : मध्यें पुरुषु ।११६।

अयोग्यातें योग्य करीति योग्यासि परमेश्वरु ज्ञान देति ।११७।

कव्हणा कव्हणी व्यसनु : कव्हणा कव्हणी व्यसनु : एथ जीव उद्धजेंति हेंचि व्यसन ।११८।

हें इशाळुवांचे इशाळु ।११९।

मूर्ती परमेश्रु एक घेउनि एक देति : अमूर्ति काहींचि नेघति ।१२०।

दया : मया : कृपा : करुणा : हें ईश्वरींचि आति ।१२१।

अन्यत्र आति तें ईश्वरीं आति : ईश्वरीं आति तें अन्यत्र नाहीं : ईश्वरीचें तें सकळही नित्य ।१२२।

परमेश्वरु जीवां नित्यबंधु ।१२३।

जीवेश्वरा स्वामीभृत्यसंबंधु हा अनादीचा ।१२४।

जीवाचां बंधमोक्षीं परमेश्वरु क्रीडति ।१२५।

जेंचि जीव जीवापौनि पावति तेंचि परमेश्वरापासौनि पावति : एक वांचौनि ।१२६।

जीव जीवां साजात्य हें सकळांही वैजात्य ।१२७।

जेचि विघ्न तेंचि अधीन : तेयासीचि वेगळा होए ।१२८।

अपरोक्षज्ञानियासि बहुरुपता : बहुकामता : होए ।१२९।

कृपास्पदीभूत होउनि अनंतकाळ असे : परि अगाध चक्रवर्ती तेयाची वासही न पाहे ।१३०।

एकु जीव कैवल्य : एकु पर कैवल्य : जीवकैवल्य तें परकैवल्यीं रमे : परकैवल्य तें स्वस्वरुपीं रमे ।१३१।

आत्मज्ञानें मोक्षु ।१३२।

लिंगपातें मोक्षु ।१३३।

स्वभावो प्रकटे तै जीव मुंचे ।१३४।

ईश्वरीं विषयप्रेम तें दुर्लभु ।१३५।

हो कां गा : तें काइ एरातें आपजउं न सके ।१३६।

बाइ : भक्ताची थोरी देवोचि जाणें ।१३७।

भक्त आपणेयातें देवो न म्हणवी ।१३८।

अपरधर्मापासौनि काढी : परधर्मी लावी ।१३९।

पोरें हो : हें काइ सानें : हें काइ कव्हणीं ठाइं असे ।१४०।

जेंचि देवांचेया मातापितरां तेचिं भक्ताचेया ।१४१।

सन्निधानिं असन्निधान : एकां असन्निधानिं सन्निधान ।१४२।

ज्ञानपूज्य : ज्ञानवृध्द तो सर्वत्र पूज्य ।१४३।

हिरवा टुटवा जीव : ईश्वरबोधें बोधवंत तेयाची सरि चैतन्यविद्यावंत न पवे ।१४४।

एणें बोधें बोधवंता जें स्वस्थता आति ते चैतन्यविद्यावंता नाहीं ।१४५।

एकु वैराग्यें उकिरडां लोळे : उष्टावळीयेचीं सीतें वेंचीं : परि पडिलीये म्हैसी प्रमाण नेणें : एकुहस्तीचा गजस्कंधीं जाय : कनकदंडी विजयमान : परि परधर्मु तो तेयातेंचिं पुसावा कीं ।१४६।

जें जैसें असे तें तैसें जाणिजे तें ज्ञान ।१४७।

सामान्य ज्ञान : खंडज्ञान : विज्ञान : विशेषज्ञान : अखंडज्ञान ।१४८।

वृत्तिरुपज्ञान : रसरुपज्ञान : विषयरुपज्ञान ।१४९।

हें शास्त्र हंसावतारें ब्रह्मेयांप्रति निरुपिलें : परि अधिकारेंविण घेतें द्वार नुघडेचि ।१५०।

सकळही शास्त्रें या शास्त्रासि मिळेति : परि हें कव्हणाही न मिळे ।१५१।

हें हाट चौहाटां बोलावें : परि एथौनि दीधलेया विण कव्हणाही घेवों घोटों न ये ।१५२।

तुम्हां विचारशास्त्रचरितार्थ उपदेशाधिकारीए ।१५३।

तुम्हां पेंडिया भरु मुद्रिला ।१५४।

वानरेशां ज्ञान झीन्न ।१५५।

एथौनि वानरेशां काहीं एकु न होआवें : परि पोरा जीवासि विज्ञान देओं न ये ।१५६।

आचार्या नरक ।१५७।

तैसा तूं परतारकु आचार्या नव्हेसि कीं गा : तूं प्राणियांतें अनिष्टा पासौनि काढौनि इष्टी लाविता आचार्या कीं ।१५८।

तुझेनि मुखें अशास्त्रीय न निगे कीं ।१५९।

बाइ : हा काहीं आपुलें बोलैल : हा एथिचेंचि बोलैल कीं ।१६०।

वानरेया : इतुले दीस काइ करितां काइ होत असे : एसें जाणा ना गा : तुम्हां अनंता सृष्टी अतिक्रांता ।१६१।

तुम्हीं पुराणीं पढीये जाल ।१६२।एथौनि तुमची थोरी सांघिजैल तरि माथेया डोळे जाती ।१६३।

जवं जवं जाणता तवं तवं नेणता ।१६४।

बाइ : जाणतेया पसि वरि नेणता चांगु ।१६५।

वोलेया पसि कोरडा चांगु ।१६६।

तर्कु अप्रतिष्ठ : तर्के अर्थ वितर्के : परि यथार्थु न भंगे ।१६७।

अहेतुक भूतभजन ज्ञानाप्रति साधन ।१६८।

अहेतुक मायापुरभजन प्रेमाप्रति साधन ।१६९।

माया संबंधु : मायाभजन ।१७०।

वस्तुसंबंधें शक्तिनिक्षेप ।१७१।

वस्तुसंबंधें तिर्यंचा योनि मोक्षु : देवां आनंदु ।१७२।

वस्तुसंबंधें जडां अजडां भलें ।१७३।

आधि राइकें विके ।१७४।

बलवत्तरात् दुर्बलं बाध्यते ।१७५।

लीळादान : वस्तुसंबंधें : एक घेउनि एक देति : कैवल्यदान ।१७६।

दृक् : स्पर्शु : आलापु : अंत:करण।१७७।

गुरुबुद्धि : देवताबुद्धि : विशिष्ठबुद्धि ।१७८।

कर्मजडापसि भक्तिजड चांगु ।१७९।

कर्मवासनेपसि ब्रह्मवासना उत्तम ।१८०।

जोगियासि प्रतिदेह नाहीं ।१८१।

भावैसणीं जोगियाचा सो सुये तेयाहीं एक भलें आति ।१८२।

मातापितरांचेनि अभिमानें बाळकासि भलें ।१८३।

मातेचें स्नेह तें नैसर्गिक ।१८४।

परमेश्वरीं जीवाचें तेयाचा एक देशु मातेचां ठाइं वर्ते ।१८५।

स्नेहाचां ठाइं उचित स्फुरे ।१८६।

स्नेहाचा कांजी जाणावे ।१८७।

जेचि एरें ठाइं वर्ते तेंचि एरें ठाइं वर्ते ।१८८।

जीव जीवां उरले असति ।१८९।

जीवु स्वार्थपरु ।१९०।

दु:खरुपता : पापमूळता : अनित्यता ए तीन्हीं नव्हेति तरि संसारावांचौनि आणिक काहीं गोमटें असे ।१९१।

जातिविखो एकु आति : तो सेविलेयां तुष्टि : पुष्टि : रस : श्रेय : इये च्यार्‍हि होति जैसें गोघृत ।१९२।

जाति विखो सेविलेयां इंद्रिय काठिन्य निवर्ते : परि अंत:करण काठिन्य न निवर्ते ।१९३।

यंत्रारुढा होइजें तो उसीटा विखो भोगणें कीं : विखो तो हावभावींचि भोगों सरे ।१९४।

विखो काइ एक मुख विखो तो सर्वतो मुख कीं ।१९५।

भक्त : योगिया : विषया : यां तीहीं धातु साम्य आति ।१९६।

दंडी : दर्शी : स्पर्शी : कुचमर्दनी ।१९७।

स्त्री सहसा प्रवर्ते ना : प्रवर्तली तरि निवर्ते ना : पुरुषु प्रवर्तो जाणें : निवर्तो जाणें ।१९८।

स्त्रियांसि अष्टगुणा कामु :परि लज्जारुप धर्मु एकु आति ।१९९।

स्त्रिया अविद्यात्मका ।२००।

स्त्रियांचां स्वरुपीं आग्रहो एकु आति ।२०१।

जीवु सर्व दोषांचें आळें ।२०२।

एथौनि तुमचे दोेष बाहिरि घालिजेति तरि पोरे हो पोटीं सुरी सों सों मराल ।२०३।

व्यवहारु तो व्यवहारेंचि सिद्धि जाए कीं गा ।२०४।

कर्मवशें जीव सकळही पावे : एक वांचौनि ।२०५।

क्रिया उभयरुपा ।२०६।

जीवु क्षेणा एकां माजी तें एकु आपजवी आपणेयांसीं जें जन्माचां सहस्त्रीं निस्तरों न सरे ।२०७।

कर्मभूमीं एक करीतां एक निमे : एक निफजे : स्वर्गी नरकीं असे तेंचि नीमे ।२०८।

देखिलें गा कें जोन्हळेयांचें पाणी आणि के घामोडेयाची साली आणि के ब्रह्महनन परि हा ठावों वेर्‍हीं मत्तद्रव्य दुषिलें ।२०९।

हिमासेतुमध्यें पांच शतें कर्मभूमि ।२१०।

जीव आर्जक : देवता फळदाती : चैतन्य व्यापारतें : परमेश्वरु उद्धरीता ।२११।

अनंतें ब्रह्माण्डें : अनंता देवता : अनंत जीव : अनंत परमेश्वर ।२१२।

कृमींतें ब्रह्मा कीजे ऐसी श्रीप्रभुची लीळा ।२१३।

नेणिजे श्रीप्रभुची लीळा कव्हणींए वेळीं कव्हणा जीवा काइ देति ।२१४।

पोरें हो तेथ काइ काहीं नेणिजे ऐसें असे : तेथ सकळही जाणौनिचि नेणिजे कीं ।२१५।

ढोरें राखा हो नाएका : तेथ आणि तेवीचि आणिकां सांगावें लागे ।२१६।

तेथिचे सकळही तें तेथिचेयाचि प्रवृत्ती ।२१७।

तेथ समळां जीवां अधिकारु नाहीं ।२१८।

ऐसेया जीवां श्रीप्रभूची सेवा : परि नेणिजे ईश्वरीचीप्रवृत्ति ते कैसी ।२१९।

एथ आणि श्रीकृष्णीं त्रिविध जन : श्रीप्रभुचां ठाइ द्विध जन।२२०।

ऐसे गा ते वेधाचार्य ।२२१।

बाइ : एथौनि जरि अव्यवस्था होआवी तरि व्यवस्था ते कोठौनि होआवी ।२२२।

अनंतशक्ति परमेश्वरु सकळांसिही विषयव्यवस्था करीति ।२२३।

कें अचिन्त्यानन्तशक्ति परमेश्वरु : आणि कें चिंतामणि : कामधेनु ।२२४।

हां गा : देव काहीं अनेक असति : देवो तो एकचि कीं ।२२५।

बाइ : अनंतु तो अनंतु कीं ।२२६।

देवो तो अच्छेद्यो : अभद्यो ।२२७।

बाइ : जीव जीवार्चन घडे परि ईश्वरार्चन तें काहींचि न घडे ।२२८।

पूर्वी म्हातारियेसी ईश्वरसेवा घडली म्हणौनि घरा जाउनि संबंधु दिधला ।२२९।

बाइ : भावो अपूर्व की ।२३०।

आउवेचा भावो काइसेया एका न पुरे : वासना प्रेमा वारिया सरे ।२३१।

एकाचें आइकत खेवों जाए : एकां अवस्था प्रकटे : एकु उपायें करोनि धाडी : सुरी : पाटी : विष : वाघवळ : हेंचि जरि ज्ञानिया करी तरि आत्महननरुप मळ लागे : परि वायां न वचे ।२३२।

पोरें हो : आतां मरा तरि चांगु ।२३३।

बाइ : भक्त आणि मागें ।२३४।

बाइ : तो हें शब्दे म्हणतु असे : एका ऐसेंचि होउनि ठाके ।२३५।

हां गा हे तुम्हां विषयत्वेंचि लागौनि ठाकलें मा ।२३६।

मंडळीका : तुम्हीं भाजीचिया सिया जाल : वानरेशु रस पीति आणि उडौनि जाति ।२३७।

एइल तैं तोचि एवों जाणैल कीं ।२३८।

बाइ : ब्राम्हणु कैसा नीका : जेया तेया अर्थां आइताचि असे : एथ होता तवं पुसतुचि होता : तुम्हीं देखतीति तरि तुम्हा सुखचि होतें ।२३९।

वानरेश आजिचि घडी धरौनि तुकितुचि होते : ते एथौनिचि घडिले ।२४०।

भांडारेकारां दर्शनासरिसी ज्ञात : आप्त : ऐसी प्रतीति अवधारिली ।२४१।

ॐ म्हणौनि भांडारेकाराचेया सारिखें उठों बैसों कव्हणींचि नेणिजे ।२४२।

बाइ : तुम्हीं राजकुमरें कीं ।२४३।

बाइ : तुमचा तो नित्यपति कीं ।२४४।

बाइ : तुम्हीं माये कीं : मायेचां ठांइ आणि एथ सकळही सामाये कीं ।२४५।

पोरें हो : जैसे मासे तैसे तुम्ही : एथौनि तुमची थोरी सांघिजैल तरि माथेया डोळे जाति ।२४६।

बाइ : एथ याचाअभिमानु : मा हें काइ याप्रति बोलिलें : हें याप्रति बोलिले कीं ।२४७।

बाइ : आधि एथ बुद्धि होआवी : मग एथचेया परिवारा वरि होआवी ।२४८।

बाइ : एथ आलेयां तेथिचिया इतुली ही श्रमनिवृत्ति न होति ।२४९।

बाइ एथही आलेयां आणि प्राणियें जळतां जावें : तरि निवावें ते कें।२५०।

सन्निधानीं जें घडले वोखटें तें सन्निधानींचि भोगितां परि कैसेंही अन्यथात्व नव्हे ।२५१।

अन्यत्र कीजे तें वाराणसी फेडिजे : वाराणसी कीजे तें कें फेडावें नाएका ।२५२।

एथिचेनि सन्निधाने कर्मे नासा ना तरि कें नासाल नाएका ।२५३।मळ कर्मे निवर्तलेयांविण जीवु कृपेसि पात्र नव्हे ।२५४।

अग्निसंबंधें वात : श्लेष्मा न बधीति कीं गा मंडळीका ।२५५।

एकी आफळ साफळ करी : एकीचेनि नावें तो विखोचि नव्हे ।२५६।

जरठांधर्मी अधिकारु नाहीं ।२५७।

वेडेयां पिसेया धर्मी अधिकारु नाहीं ।२५८।

व्यवहारीची बुद्धि धर्मी उपयोगा जाए ।२५९।

आठां वरुषा विकल्पु संचरे ।२६०।

परमेश्वरापासौनि विद्या होए : का विद्यावंता पासौनि होए ।२६१।

ऐसें हें एथीचेया नामापासौनि होए : कां स्थानापासौनि होए।२६२।

हें हों जाणें : परि जावो नेणें ।२६३।

बाइ : बिढारावरि काइ बिढारु असे ।२६४।

बाइ : शांभवीचा रश्मि होए तरि सकळही आत्मत्वें देखें ।२६५।

वशिष्ठ आणवीचिये प्रथम करणीची रीगवणी जाणें : तेयातव देहद्वन्द्वें निवर्तलीं परि मनोद्वन्द्वें न निवर्ततिचि : पुत्रवियोगें आडी दरडी वरौनि घाली परि अभिमानिनी देवता वरिचेयावरि धरी : मरो नेदी ।२६६।

सृष्टीमध्यें जेतुलेही जीवु करी तेतुलेनि काइसेनि ही मुंचे ना : एक वांचौनि ।२६७।

संसारु म्हणिजे हा दीर्घ स्वप्न गा ।२६८।

अज्ञानाचेनि प्रसादें हा जनु सुखिया होउनि असे ।२६९।

एथौनि देवताचक्र निरुपितां पृथ्वी एसणें कडीत आणि मेरु एसणी खडी न पुरे ।२७०।

तेयांचां ठाइं एके सामर्थ्य आति : तियें तरि एथौनिचि सिद्धि नेइजति कीं ।२७१।

बहुरत्ना वसुंधरा ।२७२।

शिष्याभिमानु तो म्हेळियेचेयापसि अधिकु।२७३।

बाइ : दाहा दीसां भीतरी कीजे तें पावें ।२७४।

हा प्रतीति पंथु कीं : देमति ।२७५।

बाइ : एथिचिए आने हे समर्थ कीं ।२७६।

बाइ : श्रीउद्धवदेवो आज्ञाभंग करौनि आले तरि काइ तेयांसि अविधि जाला : मा ईश्वरीची आज्ञा ते ईश्वराचिकारणें भंगिली कीं ।२७७।

उपाये न पविजे ऐसें काइ असे ।२७८।

एकु शतवेधिया रसु : एकु सहस्त्रवेधिया रसु : एकु शब्दवेधिया रसु : ।२७९।

मणिमंत्रौषधिप्रतिबंधकत्वें प्रमाणआति ।२८०।

दर्शन : अनुसरण : सेवा : संबंधु ।२८१।

श्रीदत्तात्रेय प्रभुचा चतुर्युगीं अवतारु ।२८२।

अलर्काचें तीन्हीं ताप दर्शनासरिसे शंबली शंबली म्हणौनि फेडिते जाले : कां तें अमृतवर्षिणी कीं ।२८३।

ते अमोघ दर्शन कींगा ।२८४।

या मार्गासी श्रीदत्तात्रेयप्रभु आदिकारण ।२८५।

विचार माळिका

ज्ञान मोचक ।१।

नित्य व्यापक परमेश्वर ।२।

अप्रमेयु ।३।

अनिमित्त बंधु ।४।

अनाथनाथु ।५।

कृपाळु ।६।

आर्तदानी ।७।

परमेश्वरपुरुषार्थु : लाभापासौनि लाभु ।८।

परमेश्वरु सेव्यो : जीवु सेवकु ।९।

विरुद्धा धर्मी समावेश ईश्वरींचि आति ।१०।

आपण करीना : आणिका करवी करवीना : करीतां तरि सकळहीं देखे तो साक्षी बोलिजे ।११।

त्र्यंशु तरि एकु कैसा : ना तेथ एकु वेगळा करों न ये : म्हणौनि तेथ अंशु अंशिया एैसा काहीं भावो असे : त्र्यंशु हा निरुपावेया दोषें म्हणिजति असीजे : वासना निरुपावेया दोषें म्हणिजे : तें सकळांचें ज्ञान होआवेया ।१२।

ब्रह्म तें अस्ति नास्ति ऐसेयासि विखो नव्हे ।१३।

ब्रह्म सकळधर्महित : माया धर्मवंती : ईश्वरु सकळधर्मरहितु होए आणि युक्तही होए ।१४।

शक्तिंतें अधिकरौनि परमेश्वरुचि व्यापारति ।१५।

अव्यक्तीं व्यापारु नाहीं ।१६।

प्रतिसृष्टीं परमेश्वरु अवतरेति ।१७।

अनंत अवतार ।१८।

परमेश्वर अनंतीं अवतारीं अवतरले असति : आतां अवतरले असति : साउमे अवतरति ।१९।

परमेश्वरु परदृश्य होति : अवरदृश्ये होति : परावरदृश्ये होति ।२०।

चहु युगीं परमेश्वरु जीवांतें उद्धरीति ।२१।

अनंत उद्धरिले : अनंत उद्धरीजति : अनंत असतुचिअसति ।२२।

वैखरीया अस्तित्वें परमेश्वरु तीहीतें अनुवर्तवीति ।२३।

जवं पुरुषु विद्येअधिन नव्हे तवं ब्रह्मविद्ये अधिकारु आति ।२४।

बोधकाळीं देवता परतीया सरेति ।२५।

बोधकाळीं मळकर्में परतें सारीति ।२६।

बोधु देतिए अवसरीं जींवु च्यारी घडिया पर्यंत अनाथु होउनि असे : मग केतुले एक दीस दैव वाट पाहे मग उदासीन होए ।२७।

ज्ञान : भक्ति : वैराग्य : इयें तीन्ही एके काळीं न होति ।२८।

ज्ञान : भक्ति : वैराग्य : एकें मोचकें एकें अमोचकें ।२९।

ज्ञानापसि प्रेम उत्तम ।३०।

बाइ : भक्त आणि सेजां गांवीं ।३१।

बाइ : आत्मा आणि गीतीं गाइजे : तेयातें कव्हणीचि नेणें कीं ।३२।

बाइ : मृताचे धर्म जिवंता प्रति कैसे निरुपावे ।३३।

जीवाचां माथां अनादि अविद्या असे गा ।३४।

जीवु विषम परमेश्वरु सम ।३५।

जीव आणि बांधावे : आणि सोडावे : ऐसेयां जीवां श्रीप्रभु मोकलीति तैचि मोकलीति ।३६।

बाइ : जीव जीवां काइ होति ।३७।

परमेश्वरु एकु सोडविता ।३८।

जीवु जैसा संगी ठेविजे तैसा ठाके ।३९।

मर मर शोच्या जीवा : तूते काइ असे : मा काइ देसि ।४०।

ऐसे गा जीव : मरौनि ऐश्वर्य साधिति ।४१।

जीवु म्हणिजे दोषदर्शिया गा ।४२।

जीव जीवाचें म्हणितलें करीति ।४३।

चहु युगीं हिंसा अधर्मु।४४।

तिये युगीं द्वंद्वावस्थान ।४५।

तिये युगीं अंतीं विकारु ।४६।

तिये युगीं मनोद्वन्द्वें आति : परि देहद्वन्द्वें नाही ।४७।

सकळही साधनें तीहीं युगीं वर्तती ।४८।

परमेश्वरु स्वयं विद्या स्वीकरीति ।४९।

द्वारावतीकारां पासौनि अनंता सिद्धि ।५०।

सूपेनि मार्ग : खराटेनि विद्या प्रकटेंति ।५१।

दुंडी राउळां मंत्रजाति विदीत ।५२।

बाइ : तेथौनि ऐसेयां बावना पुरुषां विद्यादान ।५३।

इंद्रियद्वारकु शक्तिसंचारु ।५४।

देवता ने : आणि : करी : करवी : आपुला भक्त ऐसें म्हणे ।५५।

विद्यावंत पुरुषु होए : तरि सृष्टिसंहाराकरीता समर्थु नव्हे ।५६।

सहस्त्र वर्षे आयुष्य : रायें ईश्वरत्वें मानीति ।५७।

त्रिकाळ ज्ञान ।५८।

एकी विद्या राजसा : एकी तामसा : बाइ : ऐसिया एकी विद्या असति जेथौनि होति तेथचि प्रतिकुळ करावें ऐसी बुद्धि उपजे ।५९।

विद्ये निरंतर फळ ।६०।

जेचिं अंतर्यागे पाविजे : तेंचि बहिर्यागें पाविजे ।६१।

निमित्तातें साक्षी करौनि देवता विद्या दे ।६२।

स्वर्गी गत्वा मुक्ति एकि आति ।६३।

परलोकीची देवता एउनि ब्रह्मविद्या उपदेशु करी ।६४।

लीळावशें पशुसि परमेश्वरु विद्यानंद देति : का फळानंद देति ।६५।

अनार्जित ते लीळा ।६६।

श्रीप्रभूपासौनि कैवल्यदान ।६७।

उदासीन वस्तु ते : तेथ काहीं काई कार्य हेतु असे ।६८।

विदेही पुरुष ते : तेथ तुम्हां असावेया अधिकारु नाहीं ।६९।

देहविद्या एकी आति : ते ईश्वरापासौनिचि होए ।७०।

देहविद्ये भूतें तन्मात्ररुपें होति ।७१।

देहविद्ये देहचि फळ ।७२।

एकी पुरुषी विद्या ते ।७३।

ऋद्धि सिद्धि होति ।७४।

भूतें पर्यंत गमन दर्शन ।७५।

अवांतर संहारु एकु आति : तो महा संहाराचि सारिखा : तेयाचां ठाइं तम एक आति ।७६।

भूतें पर्यंत संहारु यातें अवांतर संहारु बोलिजे ।७७।

अष्टभैरवांचा ठाइं पांचै भूतें : तीन्हीं गुण ।७८।

वशिष्ठ आणवीचें प्रथम करण : शुक्र अर्ध करण ।७९।

शुकाची ते चैतन्य विद्या ।८०।

चैतन्यविद्या आदिकरौनि सकळही विद्या अपरा अमोचका बोलिजेति : चैतन्यापरु परमेश्वरु पाविजे तें अच्युतपद बोलिजे ।८१।

बळिभद्र देवो तो चैतन्याचा अवतारु ।८२।

भीष्मदेवो तो आठां वसु आंतुल एक वसु : परि शापातव अध:पात जाला ।८३।

क्षीराब्धीं अविक्रियु ब्रह्म की ।८४।

बाइ : नृसिंह भूत एक अवतरलें : तेयाचा कोपु ब्रह्मांडी न सामाये ।८५।

स्वर्गी गाइचा लोकु एकु आति : तेथिचि सुरभि देवता ।८६।

अंतराळी गणगंधर्व ।८७।

यक्ष यक्षिणी ।८८।

स्वर्गी इंद्रादिकांचा ठाइं विषय मानसिक विखो ।८९।

स्वर्गीचेया पसि पाताळीचा भोगु उत्तम ।९०।

जैसा तुम्हां श्वान शुकराचा विखो तैसा इंद्रादिका तुमचा ।९१।

वरिलीचिया इच्छा तळिलीचा व्यापारु ।९२।

देवता एकी ऊचा : एकी नीचा : पैली आठ : ऐली तीन्हीं समानचि ।९३।

देवता एकी राजसा : एकी तामसा ।९४।

देवतेसि खेदु आल्हादु आति ।९५।

देवतेसि कर्मलेपांचें ज्ञानआति ।९६।

देवतेसि आपणया खालौतें अपरोक्ष ।९७।

देवतेचें मळाधार ज्ञान ।९८।

देवतेचा जैसा वर्ण : जैसी वस्त्रें : जेतुलीया भुजा : जैसीं आयुधें : तैसीं प्रकाशे ।९९।

पाला पाणी आति तवं अभिमानु आति ।१००।

क्रिया प्रयुक्तीं अभिमानु ।१०१।

क्रिया क्रियेप्रति कारण : कां फळाप्रति कारण ।१०२।

क्रियेतवं लेपु लागे : एकी निर्लेपा क्रिया ।१०३।

जेंचि देहारंभक तेंचि भोगारंभक ।१०४।

पापाये फळ दु:ख : पुण्याये फळ सुख ।१०५।

अन्नदान तें उत्तम ।१०६।

कैलास वैकुंठपर्यंत तापस्पर्श एकु आति ।१०७।

पुराणें क्षीराब्धिपर्यंत देखति : आगम अष्टभैरवपर्यंत देखति ।१०८।

बाइ : गीता श्रीकृष्णोक्ति : एर अवघी व्यासोक्ति ।१०९।

बाइ : श्रीकृष्णावतारु तो गर्भीचा ।११०।

श्रीकृष्णचक्रवर्ती सहस्त्रवरिखें राज्य केलें ।१११।

माया ईश्वरीचें प्रथम करण ।११२।

सकळ ब्रह्मांडी एकीचि माया ।११३।

अज्ञानासि माया संबंधु आति ।११४।

गर्भस्थितासि विद्या आति ।११५।

जें असे तेंचि देहांतीं प्रकटे ।११६।

सहस्त्र विंचुवांची वेदना एके रोमेचा ठाइं तें दु:ख देहांतीं जीवासि होए ।११७।

विकल्पु आति तेयासि प्रतिदेहो आति ।११८।

प्रतिदेहा पासौनि निवर्तलेयां मग अतिवाहिकें फळासि नेति ।११९।

अपमृत्यु एकु आति ।१२०।

विश्वहनना तव नित्यनरक ।१२१।

विश्रांत चैतन्य जेथ असे तेथ हनन घडलेया विश्वहनन होए ।१२२।

तेया हनना परौतें हनन नाहीं : तेया भजना परौतें भजन नाहीं ।१२३।

तेया हननीं सद्रुपता नाहीं : तेया भजनीं असद्रुपता नाहीं ।१२४।

उपाओ न पविजे ऐसें कांहीं असे।१२५।

उपयोगा न वचे ऐसें कांहीं असे ।१२६।

मंडळीकु एथिचा अवयवु कीं : याचा एथ काहीं निरोधु असे ।१२७।

पुुरुषापसि स्त्रिया स्नेहाळुवा ।१२८।

स्वप्न लटिका : सुचा ते साच ।१२९।

सत्यानृत्य व्यवहारु ।१३०।

कळियुगीं सुविद्या न स्फुरेति : कुविद्या स्फुरेति ।१३१।

गंगातीरीची म्हातारी : ना वराडीची सामान्य स्त्री ।१३२।

वाराणसी म्हणिजे स्वर्गखंड : तेथ विधिनिषेध आति परि पुण्य पाप नाहीं ।१३३।

ये ब्रह्मांडीं हा प्रपंचु : आणिकीं आणिकु : ऐसीं अनंतें ब्रह्माण्डें तेथिची व्यवस्था ते अनसारिखी आणिकीचि ।१३४।

अर्थु तो प्रकरणवशें : प्रकरणें अर्थविनियोजकु : अर्थी द्दष्टांतु दीजे ।१३५।

बाइसाचां ठाइं त्रिदोष एक होता : तो सन्निधान देउनि फेडिला : मग तिसरां दीसीं कळिका एकी संचरीली ।१३६।

एथिचेनि अभिमानीयें अभिमानु सांडिला गा : आतां तुझेनि उद्यमें काइ होइल ।१३७।

काळेंकरोनि या राष्ट्रासी समुळंचि विनश्यंति ऐसें होइल गा ।१३८।

आतां नवेयां संबंधु ।१३९।

साधनापसि साध्य उत्तम ।१४०।

जीवीं सामग्री असली तरि ईश्वरी उपयोगा न वचे ऐसें काहीं असे ।१४१।

तेथौनि जीवां कैवल्यदान आति ।१४२।

परमेश्वरु शोधकु : वेधकु : बोधकु : द्रावकु ।१४३।

परमेश्वरापासौनि चतुर्विध दान आति ।१४४।

पुण्ये विण पुण्य फळ ।१४५।

परमेश्वराचा सकळही व्यापारु शक्तिद्वारकु ।१४६।

ताल्वोष्ठपुटव्यापारें यंत्रोद्भवा वैखरी ।१४७।

साक्षात्कारें देति कां अधिकरणद्वारें देति ।१४८।

पुरुषाद्यमार्ग अनंत : दैवाद्य मार्गु तो एकु ।१४९।

अज्ञान : अज्ञानीं अन्यथा ज्ञान : अन्यथा ज्ञानीं ज्ञान : निद्रा : स्वप्न : स्वप्नीं आत्मज्ञान ।१५०।

जीवु हंसु : परमेश्वरु परमहंसु ।१५१।

विभुत्व ।१५२।

सकळही सामर्थ्ये शक्तिशब्दे बोलिजेति ।१५३।

विकरतुचि न विकरे ।१५४।

आनंदें दुमदुमित असे : ब्रह्म तें नि:शब्द ।१५५।

ब्रह्म तें सत्य : नित्य: संत : शाश्वत : सर्वधर्म शून्य : ऐसें तें ब्रह्म ।१५६।

आद्या शक्ति : माया : गुणवंती : धर्मवंती : विकारवंती : ऐसी ते माया ।१५७।

देवतास्वरुपे : ज्ञाप्तानंदरुपें ।१५८।

दुख:मिश्रित सुख तें सुख बोलिजे : निरतिशय सुख तो आनंदु बोलिजे ।१५९।

देवता नीचा उचेतें देखणें नाहीं : ऊचेचेया प्रवृत्ती नीचेचा संहारु ।१६०।

पापें विण पाप फळ ।१६१।

प्रपंचु अनित्य ।१६२।

शता वृश्चिकांचेया इतुली पीडा एका सर्पाची होए : शता सर्पांची पीडा एका देहांती जीवासि होए ।१६३।

जीव : देेवता : प्रपंचु : साजात्य : नाम साम्यें साजात्य ।१६४।

जीव : देवता : सापेक्षा : परमेश्वरु निरपेक्षु ।१६५।

जीव : देवता : साभिमानीया : परमेश्वरु निराभिमानु ।१६६।

जीव भजक : देवता भज्या ।१६७।

इंद्रिया मनातें चेष्टविता आत्मा : सकळा ही प्रकृतीतें चेेष्टविता परमात्मा ।१६८।

कर्मभूमीं विधि लेपु : भावो नाहीं : अन्यत्र विधि नाही : भावो नाहीं ।१६९।

युग पुरे तवं अवगळा असे ।१७०।

आयुष्याचा अंती विकार निमित्ते देह पात ।१७१।

साधनीं दु:खरुपता : साध्यीं सुखरुपता तो बहिर्यागु बोलिजे ।१७२।

वज्रोळी : अमरोळी : सिद्धोळी : दिव्योळी : या चार्‍ही ओळी नाथाचिया : कलियुगीं वज्रोळी : अमरोळी आति : सिद्धोळी : दिव्योळी नाहीं ।१७३।

हे चार्‍ही अष्टभैरवांचे आगम बहिर्याग ।१७४।

अष्टभैरवांचां ठाइं दिग्देशु : काळु : लोकु : नाहीं ।१७५।

एथीचेया पसि स्वर्गीचा विखोे उत्तम : तो हावोभावीचि भोगिजे ।१७६।

लौकिक मुक्ती स्त्रिया आति परि यंत्रारुढता नाहीं ।१७७।

सालोक्य : सामीप्य : सारुप्य : सायुज्य ।१७८।

इंद्रादिकांचीं फळें तो स्वर्ग बोलिजे : कैलास : वैकुंठ तो मोक्ष बोलिजे ।१७९।

लोकिक तें लोकिकीं वर्ते : भौतिक तें अष्टभैरवीं वर्ते : उपसंहाराख्य ते विश्वचैतन्याचेया ठाइं वर्ते : लाभाख्य तें ईश्वरीं वर्ते : लाभाख्य तें द्विध : भक्ताचेया ठाइं वर्ते : ज्ञानियाचेया ठाइं वर्ते ।१८०।

साध्य साधनीं सुखरुपता तो अंतर्यागु बोलिजे ।१८१।

चतु:प्रकारें वेध संचरेति ।१८२।

विद्यावंता ईश्वरीं पुरुष ऐसी प्रतीति ।१८३।

आणव : शाक्तेय : शांभव : ऐसा त्रिविध वेधु ।१८४।

पिंडीं वेधे ब्रह्माण्डीं प्राप्ति ।१८५।

अदंडी आणवीची प्रथम करणीची रिगवणी जाणें ।१८६।

चक्षुवेधें सूर्याची प्राप्ति ।१८७।

जेतुल जेतुले पर प्रकाशे : तेतुल तेतुलें अवर प्रकाशे ।१८८।

शांकरी : शांभवी : शुद्ध : वेधवंती : ब्रह्मा : विष्णु : महादेव : शेषशय्या : या चौघाचिया च्यारी विद्या : शांकरी ब्रह्मेयाची : शांभवी महादेवाची : शुद्ध विष्णुची : वेधवंती शेषशय्याची ।१८९।

अशेष पिंडवेधें विश्वेची प्राप्ति ।१९०।

एक समळ शांभव : एक शुद्ध शांभव : समळ शांभव तें विश्वेची विद्या : शुद्ध शांभव तें चैतन्याची विद्या ।१९१।

चैतन्यासि चैतन्य विद्या : जीवासि ब्रह्मविद्या ।१९२।

चैतन्य विद्या तो चैतन्याचा मोक्षमार्गु : ब्रह्मविद्या तो जीवाचां मोक्षमार्गु ।१९३।

चैतन्य चैतन्यातें अनुभवी ।१९४।

सकळा ही विद्या चैतन्यादिभावें : ब्रह्मविद्या जीवादिभावें ।१९५।

सकळांही विद्येचीं साधनें एथ : साध्य ब्रह्मांडीं : ब्रह्मविद्ये साध्य-साधन एथचि ।१९६।

केवळ केवळेसि साजात्य ।१९७।

ब्रह्मविद्ये सकळांही विद्या अनुषंगिका : रिद्धित्वें प्रकटति : कां विघ्नत्वें प्रकटति ।१९८।

चेइलेया स्वप्न लटिका तैसा अपरोक्षज्ञानिया प्रपंचु मिथ्या।१९९।

किंकरधर्मी वर्ते तेयाचेनि सकळांचे ही संदेह तुटीति : तो म्हणे तो अर्थु तैसाचि होए ।२००।

ज्ञानियासि अनिष्ट निवृत्ति करीति : भक्तासि अनिष्ट निवृत्ति करोनि इष्टप्राप्तिकरीति ।२०१।

आनंदे दुंदुभि ।२०२।

नाहं मनुष्यो न च देवयक्षो न ब्राह्मणक्षत्रियवैश्यशुद्रा: न ब्रह्मचारी न गृही वनस्थो भिक्षुर्न चाहं निजबोधरुप: ।२०३।
